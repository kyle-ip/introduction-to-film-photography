# 入门准备

## 简介

本部分介绍胶片与数码的区别、相机与胶卷选择、必备附件及入门预算，帮助你做好开拍前的准备。

## 胶片摄影简介

### 胶片摄影与数码摄影的区别

| 特性      | 胶片摄影                                                     | 数码摄影                                                     |
| --------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 工作原理  | 使用化学感光材料，光线照射卤化银晶体形成潜影，通过显影定影成为可见影像 | 使用电子传感器（CMOS/CCD），将光信号转换为电信号，经处理器生成数字文件 |
| 即时反馈  | 无，需冲洗后才能看到结果                                     | 有，拍摄后可立即查看 LCD 屏                                  |
| 成本结构  | 持续成本：胶卷、冲洗、扫描                                   | 主要是前期设备投入                                           |
| 色彩表现  | 独特的色彩科学，不同胶卷有不同色彩倾向，自然过渡             | 色彩可后期大幅调整，取决于传感器和处理器                     |
| 颗粒/噪点 | 胶片颗粒，有质感，高 ISO 颗粒增加                            | 数码噪点，高 ISO 噪点明显                                    |
| 动态范围  | 通常宽容度较好，特别是黑白胶片                               | 现代数码传感器动态范围也很优秀                               |
| 后期处理  | 暗房（化学）或扫描后数码后期                                 | 数码后期，灵活度高                                           |
| 存档方式  | 底片实体存档，需妥善保管                                     | 数字文件，可多备份                                           |

### 为什么选择胶片摄影

#### 独特的美学

胶片的色彩表现是数码难以复制的，特别是柯达和富士的经典胶卷，胶片颗粒带来独特的质感而不是数码的噪点，对高光和暗部的表现也有其独特方式。

#### 慢摄影

每张都有成本，促使你在按快门前确认构图、曝光、对焦，从而减少废片、扎实基础；没有即时回放，反而有利于养成严谨习惯。

#### 仪式感

从拍摄、装卷、冲洗到扫描或放大，完整流程带来参与感；等待冲洗与亲手冲出的第一张底片，都充满期待与满足。

#### 收藏价值

底片可长期保存（数十年至上百年），具有收藏与史料价值；许多经典摄影作品均出自胶片。

#### 学习体验

胶片摄影让你更深入地理解曝光、显影等基本原理，可以实践安塞尔·亚当斯的区域曝光法，学习传统暗房技术，理解影像形成的化学过程。

## 相机选择

选购思路与决策树、按用途的快速推荐见附录《设备购买指南》。下面先介绍相机类型与代表机型。

### 胶片相机类型

#### 135 相机（35mm）

135 相机是入门的推荐选择。其画幅为 36x24mm，即全画幅。

**特点**：便携、胶卷易购（一卷约 36 张）、镜头焦段齐全，适合日常、旅行、人像及学习摄影。

**入门推荐机型详解**

- **佳能 AE-1 / AE-1 Program**：AE-1 于 1976 年推出，产量大，市面容易找到，价格实惠。AE-1 具备快门优先自动曝光功能，用户设定快门速度后，相机会自动选择光圈。AE-1 Program 增加了程序自动曝光，使用更便捷。AE-1 使用佳能 FD 卡口镜头，选择丰富且价格实惠。500 元左右可购买不错的 50mm f/1.8 大光圈镜头。AE-1 采用电子快门，电池耗尽后无法使用，建议多备电池。机身参考价格约 500-1500 元。
- **尼康 FM2 / FE2**：FM2 全机械（仅测光需电）、最高 1/4000s 快门（后期版为钛帘）；FE2 电子版、光圈优先。AI/AIS 卡口镜头素质好。FM2 约 1500-3000 元，FE2 约 1000-2000 元。
- **奥林巴斯 OM-1 / OM-2**：体积小、取景器亮。OM-1 机械快门，OM-2 电子+OTF 测光。OM 卡口镜头较少但实惠。机身约 800-2000 元。
- **宾得 K1000**：全机械学生机，手动快门/光圈/对焦，PK 卡口镜头多且便宜，适合打基础。机身约 500-1200 元。

#### 120 相机（中画幅）

120 中画幅相机画幅更大，画质更好，但通常体积和重量更大，价格也更高。

120 无暗盒、有背纸。常见画幅：6x4.5 约 15-16 张，6x6 约 12 张，6x7 约 10 张。适合风光、人像、静物、商业等高画质需求；日常随身更推荐 135。

**入门推荐机型详解**

- **玛米亚 M645**：M645 为 645 画幅单反，采用模块化设计，可更换后背和取景器。有多个版本，如 1000S、Super 等，镜头选择丰富。机身加标准镜头参考价格约 1500-3000 元，性价比高。
- **勃朗尼卡 ETRS**：ETRS 同样为 645 画幅模块化设计，镜头质量良好，价格实惠，参考价格约 1000-2500 元。
- **基辅 88**：基辅 88 仿制哈苏 500 系列，采用 6x6 方形画幅。价格便宜，参考价格约 800-2000 元，但做工一般，可能存在小问题。适合体验方形构图和中画幅。

#### 旁轴/双反相机（RF/TLR）

旁轴和双反相机无反光板，快门声音安静。多数双反采用俯视取景，6x6 方形画幅是其特色。

**双反特点**：双镜头（上取景、下拍摄），俯视毛玻璃取景、画面左右反向，快门轻、方形构图。适合人像低机位、街头纪实；多数不可换镜头（玛米亚 C 除外），俯视取景不利于快速抓拍，体积较大。

**135 旁轴推荐**

- **康泰时 T2 / 奥林巴斯 XA**：两款相机均体积小巧便携，固定镜头画质良好，适合街头拍摄，不易引人注目。参考价格约 1500-3000 元。
- **徕卡 M 系列**：徕卡 M 系列为经典旁轴系统，可更换镜头，画质优秀。机身参考价格 5000 元以上，预算充足时为优质选择。

**120 双反推荐**

- **禄来 Rolleiflex 3.5F / 2.8F**：传奇双反，蔡司/施耐德镜头，全机械。3.5F 为 75mm f/3.5，2.8F 为 80mm f/2.8。3.5F 约 3000-6000 元，2.8F 约 5000-10000 元。
- **玛米亚 C330 / C220**：可换镜头双反，55/80/135/180mm 等，腰平+眼平取景，皮腔对焦可微距。机身+镜头约 2000-4000 元，体积较大。
- **雅西卡 Yashica Mat-124G**：双反入门机，带测光，80mm f/3.5，约 1000-2000 元。
- **海鸥 4A / 4B**：国产仿禄来，全机械，适合练手，约 200-500 元；做工与镜头一般。

#### 大画幅相机

大画幅相机使用单张页片，常见规格包括 4x5 英寸、5x7 英寸、8x10 英寸等。

**特点**：画幅大、画质细腻、可移轴校正透视，适合建筑与风光；操作复杂、节奏慢，不适合随手拍。入门可考虑星座（Toho）4x5；骑士、林哈夫价高。

建议先掌握 135 或 120 再入大画幅。进阶内容中有「大画幅摄影基础」小节，介绍皮腔移轴、延长系数、沙姆定律与霍夫曼定律等。

### 手动相机 vs 自动相机

#### 手动相机优势

| 优势       | 说明                                     |
| ---------- | ---------------------------------------- |
| 更可靠     | 机械结构寿命长，不易出故障               |
| 不依赖电池 | 很多纯机械相机不需要电池或仅测光需要电池 |
| 学习基础   | 迫使你学习快门、光圈、手动对焦           |
| 价格更低   | 通常价格更实惠                           |
| 操作感     | 机械快门的手感是电子快门难以复制的       |

#### 自动相机优势

| 优势       | 说明                             |
| ---------- | -------------------------------- |
| 操作便捷   | 自动曝光、自动对焦，适合快速拍摄 |
| 抓拍能力强 | 街头摄影等需要快速反应的场景     |
| 先进测光   | 有些有多点测光、评价测光等       |
| 更多功能   | 可能有包围曝光、多次曝光等功能   |

**建议入门选择**：如果是学习摄影基础，建议从手动机械相机开始；如果主要是记录生活，自动相机也可以。

## 胶卷选择

### 胶卷规格详解

#### 135 胶卷（35mm）

**基本信息**：宽度 35mm，每卷张数通常 36 张，也有 24 张、12 张装，画幅尺寸 36x24mm，有暗盒，保护胶卷不被曝光。

**特点**：便携，最普及，选择最多，相机和镜头选择丰富，后期扫描方便。

**适用场景**：日常拍摄，旅行，街头，纪实。

#### 120 胶卷

**基本信息**：宽度约 62mm，无暗盒，有背纸保护，根据相机画幅不同，可拍张数不同：6x4.5cm 15-16 张，6x6cm 12 张，6x7cm 10 张，6x9cm 8 张。

**特点**：画幅更大，画质更好，选择相对 135 少一些，相机通常更大更重。

**适用场景**：人像，风光，静物，商业。

#### 其他规格

110 胶卷是小型相机使用，画幅小且画质一般，现在少见。APS 胶卷是 90 年代推出的过渡规格，有不同画幅模式，现已少见。大画幅页片单张使用，常见尺寸有 4x5 英寸、5x7 英寸、8x10 英寸，需要片夹装载。

### 黑白胶卷特性详解

#### ISO（感光度）

ISO 表示胶卷对光线的敏感程度。

| ISO 范围     | 分类     | 特点                         | 适用场景                                 |
| ------------ | -------- | ---------------------------- | ---------------------------------------- |
| ISO 25-100   | 低速胶卷 | 颗粒极细，画质细腻，分辨率高 | 光线充足、风光、人像、静物、需要极高画质 |
| ISO 200-400  | 中速胶卷 | 颗粒适中，通用性好，宽容度好 | 日常拍摄、大多数场景、入门练习           |
| ISO 800-3200 | 高速胶卷 | 颗粒较粗，反差较高           | 低光环境、室内、夜景、需要高速快门       |
| ISO 3200+    | 超高速   | 颗粒很粗，风格强烈           | 极低光、特殊效果                         |

**选择建议**：入门练习 ISO 400（HP5+、Tri-X）最通用，户外晴天 ISO 100-200，室内/傍晚 ISO 400-800，夜景/暗光 ISO 800+或用三脚架+低速卷。

#### 颗粒度

颗粒是胶片的固有特性，来自于卤化银晶体。

**颗粒度分类**：极细颗粒几乎看不出颗粒，如富士 Acros、柯达 T-Max 100；细颗粒细腻，如伊尔福 FP4+、Delta 100；中颗粒有明显胶片质感，如伊尔福 HP5+；粗颗粒颗粒明显，风格强烈，如柯达 Tri-X 400（某些显影）、高速卷。

**颗粒度的影响因素**：胶卷本身 ISO 越高，颗粒越粗；不同显影液颗粒不同（Rodinal 以锐度著称、颗粒较 D-76 明显，D-76 适中）；显影时间越长，颗粒越粗；曝光不足会增加颗粒。

颗粒不是缺点！颗粒是胶片美学的一部分，很多人就是喜欢胶片的颗粒质感。

#### 反差

反差是指影像中最亮和最暗部分的差别。

**反差分类**：低反差明暗差别小，灰度层次丰富，宽容度高，适合大光比场景、需要后期调整；中反差明暗差别适中，通用，适合大多数场景；高反差明暗差别大，对比强烈，适合街头、纪实、戏剧化效果。

**影响反差的因素**：胶卷本身有些反差高有些低，显影液如 D-76 反差适中，Rodinal 反差较高，显影时间越长反差越高，曝光过度会降低反差。

#### 宽容度

宽容度是指胶卷能够记录的亮度范围。黑白胶卷宽容度通常很好，可达 7-10 档甚至更多；彩色负片宽容度也不错，约 5-7 档；彩色反转片宽容度较低，约 3-5 档。

**宽容度的意义**：高宽容度即使曝光有误差，也能记录下细节；大光比场景高光和暗部都能保留细节。

### 彩色反转片（正片）特性详解

#### 彩色反转片特点

彩色反转片冲洗后是正像，可直接观看，色彩饱和鲜艳，反差大，宽容度低，曝光要求精确，一般±1 档，适合投影，可以用幻灯机投影观看。

#### 常用反转片推荐详解

### 胶卷保存方法

#### 未拍摄胶卷保存

短期（几个月）常温、干燥、避光保存，长期（超过半年）放入冰箱冷藏（4°C 左右），需要长期保存（数年）可以冷冻。注意从冰箱取出后，要等恢复到室温再打开包装，避免结露！

#### 存放与送冲前准备

**尽快冲洗！最好一周内。** 潜影会随时间衰退，高温高湿会加速灰雾与彩色卷色偏。

**温湿度**：与未拍卷类似，宜凉爽、干燥、避光存放。建议温度不高于约 20–25°C，湿度避免长期高于 60%；夏季若不能马上冲，更应避免放在车内、阳台等高温处。

**存放时长**：若需延迟冲洗，可将胶卷放入冰箱冷藏（约 4°C），用密封袋防潮（必要时加干燥剂），取出后待回温再送冲或自冲，避免结露损坏乳剂。

**标记**：在暗盒或袋上标注拍摄日期、内容简述、ISO；若做了迫冲/减冲须注明，便于送冲时写单或自冲时查显影时间表。

**寄送冲洗**：若选择寄送店家冲洗，包装与寄送注意事项（避光防压、快递与保价、附纸条内容等）见附录《胶卷购买与送冲指南》中的「寄送冲洗的注意事项」。

#### 已冲洗底片保存

使用无酸底片袋，干燥、避光、常温保存，避免高温高湿，可以放入干燥剂。

#### 旅行与安检：X 光对胶片的影响

乘飞机或经过地铁、车站的安检时，行李会经过 X 光机。理解 X 光与胶片的关系，能帮助你在旅行中做出合理选择，既保护胶片又避免不必要的紧张。

**原理**：X 光与可见光一样是一种电磁辐射，但波长更短、能量更高。胶片乳剂中的卤化银对 X 光敏感，X 光照射会在乳剂中形成潜影，相当于「额外曝光」。一次安检的 X 光剂量通常不大，对 ISO 400 以下的胶卷、单次通过的影响多数情况下不明显；但多次累积（例如往返各过一次、转机再过一次）或高感光度胶卷（如 ISO 800、1600）会更容易出现灰雾或整体偏灰。未冲洗的胶卷比已冲洗的底片更敏感，因为潜影会与后续 X 光叠加；已冲洗的底片银像已固定，对 X 光仍有一定敏感度，但通常不如未拍胶卷明显。

**请求手检**：多数机场允许胶片不经过 X 光机、改由人工开包检查。你可以把未拍和已拍未冲的胶卷放在透明袋中，在安检时主动说明「这是胶片，请手检」（"Film, hand check please"），工作人员通常会取出目视检查后放行。手检不会对胶片造成影响，是最稳妥的方式。高感卷（如 ISO 800 以上）、重要拍摄或多次转机时，建议一律手检。

**防辐射袋**：市售「防 X 光」铅袋或含铅布袋能减弱部分 X 光，但无法完全屏蔽；且若机器因检测到遮挡而自动提高功率或要求开包复检，胶卷反而可能受到更强照射。因此防辐射袋不能替代手检，仅可作为辅助；若无法手检时，低感卷（如 ISO 100–400）单次通过一般风险较低，高感卷或重要卷仍建议尽量争取手检。

**实践建议**：出行前把未拍胶卷装在透明袋中、与相机分开放，便于快速取出；过安检时主动提出手检，态度礼貌、说明用途即可。若遇规定不允许手检（少数地区或场合），低感卷单次通过通常可接受，高感卷可考虑到达目的地后再购买。理解「X 光会累积、手检最稳妥」的道理后，就能根据行程和胶卷类型自行权衡。

## 必备附件清单

### 基础附件

#### 测光表

老机身测光易不准或损坏；入射式比反射式更稳，利于学习测光。

**类型**：入射式（测照到被摄体的光，乳白球朝相机）、反射式（测反射光，需 18% 灰参考）、点测光（小区域 1°-5°，适合区域曝光法）。

**推荐**：入门 Sekonic L-308S（入射+反射，二手约 500-800 元），进阶 L-758（点测）。选购看入射/反射/点测、电池与精度。

#### 快门线

长曝光与 B 门可避免手触相机、减少机震；自拍可用。机械快门线耐用、无需电池、部分带 B 门锁；电子快门线可有定时。选购看接口匹配，机械更可靠，长度 50cm-1m 即可。

#### 三脚架

用于慢速快门、夜景、长曝光、风光等。

**材质**：铝合金性价比高、较重（曼富图、百诺、思锐）；碳纤维轻便、较贵。

**选购**：承重≥机身+镜头 2 倍，展开高度不低于眼高，云台球台方便。入门百诺 A、思锐 R 约 300-800 元，中端 1000-3000 元。

#### 滤镜

- **UV 镜**：用途是保护镜头前组，过滤紫外线。选购时不要买太便宜的，影响画质，多层镀膜（MC）更好，口径匹配镜头。推荐肯高、B+W（预算够）。
- **偏振镜（CPL）**：用途是消除非金属表面反光（玻璃、水面），增加蓝天饱和度，压暗天空。选购时选择圆形偏振镜（CPL），薄框（避免暗角），推荐保谷、肯高、B+W。注意会减少 1-2 档曝光。
- **中性灰滤镜（ND）**：用途是减少进光量，在白天也能长曝光。规格有 ND2 减 1 档、ND4 减 2 档、ND8 减 3 档、ND64 减 6 档、ND1000 减 10 档（白天长曝光）。推荐先买 ND8 和 ND64 入门。
- **渐变灰滤镜（GND）**：用途是平衡天地曝光（风光常用），类型包括软渐变过渡柔和，硬渐变过渡明显，规格有 0.3（减 1 档）、0.6（减 2 档）、0.9（减 3 档）。推荐 0.6 软渐变入门。

**滤镜系统**：圆形滤镜拧在镜头上，方便，但每个镜头口径不同；方形滤镜插片式，一套用多个镜头，风光推荐。推荐系统有 Cokin、NiSi、海泰。

#### 相机包

**选购要点**：防护性有缓冲层，便携性根据你的出行方式选择，分区能分开装相机、镜头、附件，防水最好有防雨罩，容量能装下你的器材，并有一定余量。

**推荐类型**：日常/街拍选单肩包或小双肩，旅行/风光选双肩包，品牌推荐乐摄宝、国家地理、曼富图、小野。

### 额外推荐

胶片收纳盒用于保护拍摄好的胶卷，防 X 光；干燥剂放在相机包和胶卷盒里防潮；清洁套装包括气吹（首选）、镜头刷（软毛刷）、镜头纸/镜头布（微纤维）、清洁液（必要时用）；备用电池如果相机需要电池，多备几颗；灰卡 18%灰卡，用于精确测光和白平衡；笔记本/笔记录拍摄信息（ISO、光圈、快门、场景等）。

## 学习路径与第一卷建议

入门阶段最容易出现的不是「拍得不好看」，而是「不知道问题出在哪」——装卷是否成功、曝光是否合理、冲洗是否正常，都要等冲扫后才能反推。下面从原理出发，说明如何安排第一卷和后续学习顺序，让每一次按快门和每一次冲洗都变成可复盘的经验。

**为什么建议第一卷从黑白 135 开始？** 黑白胶卷只有银盐一层，冲洗流程简单（显影→停显→定影→水洗），变量少，更容易建立「曝光–显影–底片密度」的对应关系；135 画幅胶卷和相机最普及，装卷、过片、卸卷的步骤资料多、容易对照。先掌握「装卷–拍摄–冲洗–观察底片」的完整闭环，再接触彩色或 120，能避免同时面对太多未知（色温、C-41 温控、120 装卷等），挫败感更小。

**第一卷拍什么、怎么记？** 建议先拍你熟悉的场景（家里、常走的街道、静物），这样你更容易判断「这张应该是亮是暗、哪里该清晰」。每拍几张或每换一个场景，在笔记本上记下：当前张数、光圈、快门、ISO、场景简述（如「室内窗边」「阴天户外」）。不必每张都记，但关键几张（例如测光表读数和你的选择、试拍的不同曝光）记下来，冲出来后对照底片密度，就能反推「这一张是曝光过了还是欠了、显影是过了还是不足」。复盘时重点看：整卷是否都有影像（装卷是否成功）、密度是否合理（曝光与显影是否在可控范围）、有没有漏光或划痕（操作是否得当）。把「记录–冲洗–对照–归纳」变成习惯，进步会更快。

**进阶顺序建议**：在黑白 135 能稳定装卷、曝光和冲洗后，可以尝试彩色负片（送冲即可，先不自己冲 C-41），体会色温与胶卷色彩的关系；再考虑 120（画幅更大、装卷与相机操作不同，但曝光与测光原理相同）。自己冲 C-41、拍反转片、迫冲减冲等，可放在对基础流程已经熟练之后。这样每一阶段都只增加少量新变量，容易消化。

**心态**：第一卷拍糊、拍漏、冲坏都很正常，重点是「能从结果反推原因」。教程中的「装卷与拍摄常见问题」与附录《黑白胶片冲洗配方》中的问题排查表正是为这种复盘准备的——看到现象，想到可能原因，下次就能有针对性地改进。把学习路径想成「先闭环、再扩展」，就不会在一开始被太多选项压垮。

| 阶段 | 要点 |
