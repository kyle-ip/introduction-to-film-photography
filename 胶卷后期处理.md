# 胶卷后期处理

## 简介

冲洗后的底片可走两条路径：数字化（扫描/翻拍后数字后期）或暗房放大。本节先讲保存与整理，再分述数字化与暗房放大。

## 底片保存、观察与整理

### 底片保存的化学原理

#### 底片变质的原因

主要机理：① **残留定影剂**（硫代硫酸钠）与 O₂、H₂O、CO₂ 反应生成硫酸，腐蚀银粒，导致变黄褪色；② **银氧化**（Ag₂O 棕褐色），潮湿高温加速；③ **明胶降解**（高温高湿水解、发霉，酸/碱/紫外加速）；④ **片基老化**（醋酸纤维素释酸，聚酯较稳定）。

#### 理想保存条件

**温度**：每降 10°C 反应速率约减半，理想 10-15°C。**湿度**：过低明胶脆化，过高促反应与霉变，理想 30-40%。**避光**：紫外与可见光加速反应，深色容器挡光。**无酸**：酸性材料加速变质，用无酸纸与塑料。

### 底片保存的实践

#### 底片袋选择原理

普通纸含木质素会释酸，加速银氧化与明胶降解。无酸纸（化学木浆/棉浆、pH 7-8.5）可长期稳定；塑料袋用聚酯（Mylar）或聚乙烯，稳定、透明、便于查看。

#### 存档系统设计原理

**时间顺序**：按拍摄时间排，易查找、看技术进步，适合多数爱好者。**主题归档**：按项目/主题集中，便于系列与展览，适合有明确方向的创作者。**胶卷类型**：按胶卷型号集中，便于对比与学习，适合技术研究。

#### 标注信息的重要性

底片不记录拍摄参数，需人工标注。标注便于复现显影、分析失败、日后回顾与版权证明。建议标注：拍摄日期/地点/主题、胶卷类型与 ISO/批次、显影信息（药液、时间、温度、稀释）、相机/镜头、光圈快门及备注。

### 底片观察技术

#### 观片器的光学原理

**观片器**：日光色温（约 5500K）光源 + 扩散板（均匀面光）+ 消色差放大镜，便于判断密度与色彩。

#### 如何观察底片

**密度**：正常底片高光较暗（密度高）、阴影较亮（密度低）；过厚多为显影/曝光过度，过薄多为不足。**反差**：看明暗密度差；正常有丰富灰阶，过高缺中间调，过低发灰。**细节**：放大镜看阴影层次与高光细节，检查划痕、灰尘、水渍。**彩色**：色罩应均匀橙/棕，整体偏色可能为冲洗问题。

## 底片数字化原理与实践

### 为什么需要数字化

**备份**：底片易损、易丢、会变质；数字文件可多份备份（3-2-1：3 份、2 种介质、1 份异地）。**传播**：底片难分享，数字可即时网络分享、社交媒体与邮件。**后期**：数字后期可无损调整、随时撤销，软件工具丰富。**空间**：大量底片占物理空间，数字存硬盘更省。

### 扫描原理

#### 扫描仪的工作原理

扫描仪是光电转换设备，其核心原理是：

1. **光源系统**
   1. 透射扫描需要背光光源，从底片背面照射
   2. 反射扫描需要正面光源（扫描照片时使用）
   3. 光源通常是冷阴极荧光灯（CCFL）或 LED
   4. 光源的光谱特性会影响色彩还原
2. **光学系统**
   1. 镜头将底片的影像聚焦到传感器上
   2. 平板扫描仪使用移动光学组件逐行扫描
   3. 底片扫描仪使用固定光学组件，移动底片
   4. 镜头的分辨率决定了扫描仪的光学分辨率
3. **传感器**
   1. CCD（电荷耦合器件）或 CIS（接触式图像传感器）
   2. 将光信号转换为电信号
   3. CCD 通常画质更好，CIS 更轻便
   4. 传感器的像素数量决定了最大分辨率
4. **模数转换（ADC）**
   1. 将模拟电信号转换为数字信号
   2. 色彩深度（位深）表示每个通道的灰度级数
   3. 8 位=256 级，16 位=65536 级
   4. 更高的位深提供更大的后期调整空间

#### 分辨率与采样定理

理解分辨率需要了解奈奎斯特采样定理：

- **采样定理**：为了准确还原信号，采样频率必须至少是信号最高频率的两倍
- **应用到扫描**：要准确还原胶片的颗粒和细节，扫描分辨率必须至少是胶片分辨率的两倍
- **胶片分辨率**：一般黑白胶片约 100-200 线对/毫米，彩色胶片约 50-100 线对/毫米
- **实际建议**：2400dpi 适合日常使用，4800dpi 适合高质量存档，更高分辨率意义不大（收益递减）

#### 色彩深度原理

**为何 16 位**：人眼约能分辨 100-200 级灰度；调色阶/曲线会重分配灰度。8 位仅 256 级易出现断层（banding），16 位 65536 级可大幅调整仍平滑。16 位文件约为 8 位两倍，存储成本可接受。

### 扫描仪选择

#### 平板扫描仪 vs 底片扫描仪

理解两种扫描仪的光学设计差异：

|          | 平板扫描仪                                                   | 底片扫描仪                                                   |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 光学原理 | 使用大尺寸 CCD 或 CIS，配合大面积光源。                      | 使用小尺寸高分辨率 CCD，配合精确的胶片进给机构。             |
| 优势     | 可以扫描 135、120、4x5 等多种规格<br />可以扫描照片、文档等反射稿<br />性价比高，一机多用 | 光学分辨率高，通常 4800dpi 以上<br />光密度（Dmax）高，能捕捉丰富的暗部细节<br />对焦精确，画质锐利 扫描速度快 |
| 劣势     | 光学分辨率通常较低<br />光密度（Dmax）较低，难以捕捉暗部细节<br />对焦精度可能不如专用底片扫描仪 | 通常只能扫描一种规格（如 135）<br />价格较高<br />不能扫描反射稿 |

#### 光密度（Dmax）的重要性

光密度表示扫描仪能够记录的最大密度范围：

- **密度定义**：D = log₁₀（入射光/透射光）
- **D=0**：完全透明（100%透光）
- **D=1**：10%透光
- **D=2**：1%透光
- **D=3**：0.1%透光
- **D=4**：0.01%透光

底片的高光部分密度可达 3.0 以上，因此扫描仪的 Dmax 至少需要 3.5，最好 4.0 以上，才能捕捉到完整的高光细节。

### 扫描流程详解

#### 扫描前准备：清洁的科学原理

灰尘会挡光形成白/黑点，高分辨率扫描会放大灰尘；扫描前清洁比后期修更省事、少损细节。

正确的清洁方法：

1. **气吹**：使用橡胶球气吹，从侧面吹气，避免直接吹（可能将灰尘吹得更紧）
2. **防静电刷**：使用天然毛刷，带有接地线，去除静电吸附的灰尘
3. **湿清洁**：仅在必要时使用，使用专用底片清洁液和无纺布，从中心向外轻轻擦拭，一个方向，不要来回擦

#### 扫描参数设置原理

理解每个参数的作用，才能做出正确选择：

**分辨率设置**

- 原理：如前所述，基于采样定理
- 建议：
  - 网络分享：1200-2400dpi
  - 打印输出（A4）：2400dpi
  - 高质量存档：4800dpi
  - 大幅面打印（A3 以上）：4800dpi 或更高

**色彩深度设置**

- 原理：如前所述，16 位提供更大调整空间
- 建议：
  - 存档：16 位彩色（RGB 48 位）或 16 位灰度
  - 快速预览：8 位可以接受

**文件格式选择**

- **TIFF**：无损压缩，保留所有信息，适合存档
- **RAW**：某些扫描仪支持，保留传感器原始数据，后期空间最大
- **JPEG**：有损压缩，不建议存档，仅用于最终分享

**色彩空间选择**

- **sRGB**：网络分享，大多数显示器使用
- **Adobe RGB**：打印输出，色彩范围更大
- **ProPhoto RGB**：专业后期，色彩范围最大，需要 16 位支持

**除尘功能（Digital ICE）**

- 原理：使用红外线扫描，检测灰尘位置（灰尘阻挡红外线，银粒不阻挡）
- 优势：自动去除灰尘，节省后期时间
- 注意：某些黑白胶片（如含染料的）可能不支持

#### 色阶调整原理

扫描时的色阶调整基于直方图分析：

1. **直方图**
   1. X 轴：亮度（0-255，黑-白）
   2. Y 轴：像素数量
   3. 理想的直方图：左右触边但不溢出，山峰在中间
2. **黑场设置**
   1. 找到画面中最暗的地方（应该是纯黑）
   2. 将黑场滑块移动到该位置
   3. 作用：将该点设为纯黑，增加暗部对比度
3. **白场设置**
   1. 找到画面中最亮的地方（应该是纯白）
   2. 将白场滑块移动到该位置
   3. 作用：将该点设为纯白，增加亮部对比度
4. **灰场设置**
   1. 找到画面中应该是中性灰的地方
   2. 用灰场吸管点击该位置
   3. 作用：消除色偏，校正白平衡

### 翻拍原理与实践

#### 翻拍的光学原理

翻拍本质上是用数码相机给底片拍照，需要理解以下光学原理：

1. **放大倍率**
   1. 放大倍率 = 传感器尺寸 / 底片尺寸
   2. 135 底片（36x24mm）在全画幅传感器上是 1:1
   3. 在 APS-C 传感器上约 1.5-1.6 倍
   4. 微距镜头可以提供更高的放大倍率
2. **焦距选择**
   1. 焦距太短：视角太广，需要很近的对焦距离，可能产生畸变
   2. 焦距太长：对焦距离太远，需要更大的工作空间
   3. 最佳选择：50mm、60mm、100mm 微距镜头
3. **光圈选择**
   1. 衍射极限：光圈太小（如 f/16）会因为衍射而降低分辨率
   2. 景深需求：需要足够的景深覆盖整个底片厚度
   3. 最佳点：大多数镜头在 f/8-f/11 时画质最佳
4. **均匀照明**
   1. 底片需要均匀的背光照明
   2. 不均匀的照明会导致扫描件亮度不均
   3. 可以使用灯箱、两盏对称的灯、或扩散板

#### 翻拍 vs 扫描：优劣比较

理解两种方法的差异，选择最适合的方式：

| 方面       | 扫描                       | 翻拍                             |
| ---------- | -------------------------- | -------------------------------- |
| 分辨率     | 取决于扫描仪光学分辨率     | 取决于相机像素和放大倍率         |
| 操作速度   | 较慢，需要逐行扫描         | 快，按快门即可                   |
| 即时反馈   | 预览，但不如翻拍直观       | 实时取景，立即看到结果           |
| 设备成本   | 专用扫描仪较贵             | 已有相机的话，只需灯箱和微距镜头 |
| 画质一致性 | 扫描件一致性好             | 需要更注意布光和对焦             |
| 胶片规格   | 取决于扫描仪设计           | 可以灵活适应各种规格             |
| 后期需求   | 较简单，扫描仪软件处理很多 | 需要反转、去色罩等操作           |

#### 翻拍技巧详解

1. **设备布置**
   1. 相机：固定在三脚架上，确保稳定
   2. 灯箱：放在相机下方，底片放在灯箱上
   3. 确保相机与底片平行，镜头垂直于底片
   4. 使用水平仪确保水平
2. **对焦技巧**
   1. 使用实时取景（Live View）
   2. 放大 10 倍查看
   3. 对焦在底片中心
   4. 可以使用对焦峰值辅助
3. **拍摄参数**
   1. ISO：100 或 200，越低越好，减少噪点
   2. 光圈：f/8-f/11，最佳光圈
   3. 快门：根据光线调整，使用快门线或自拍模式避免机震
   4. 格式：RAW，保留最大后期空间
   5. 白平衡：使用灰卡或自定义白平衡
4. **包围曝光**
   1. 包围曝光±1 或±2 档
   2. 确保捕获完整的动态范围
   3. 可以后期合成 HDR

### 扫描与翻拍常见问题

| 现象 | 可能原因 | 对策或参考 |
|------|----------|------------|
| **牛顿环（彩色干涉条纹）** | 底片与玻璃贴合过紧、空气层产生干涉 | 扫描时用无玻璃片夹或略抬底片；翻拍时避免底片与玻璃全面接触；部分扫描仪有防牛顿环片。 |
| **顽固灰尘、划痕** | 扫描前未清洁、底片保存时刮擦 | 扫描前用气吹、防静电刷轻扫；湿清洁慎用、单向轻擦；扫描软件除尘（如 Digital ICE）对部分底片有效；严重划痕可后期局部修复。 |
| **色罩校正不理想、整体偏色** | 彩色负片色罩未正确去除、白场/灰场设错 | 用软件负片预设或灰场吸管校正；确保黑/白场设在画面中应有黑/白的区域；不同胶卷色罩不同，可存为预设。 |
| **扫描发灰、反差不足** | 黑/白场未拉到有效范围、扫描软件默认曲线偏平 | 在扫描或后期中拉准黑场与白场（直方图左右触边）；适当提高对比度或使用 S 曲线；参见本文「色阶调整原理」。 |

## 数字化后期处理原理

### 负片反转原理

为什么负片需要反转？如何科学地反转？

#### 负片的光学原理

负片是"补色反转"的：

1. **黑白负片**
   1. 拍摄时：亮部曝光多，生成的银多
   2. 底片上：亮部是黑的（银多），暗部是透明的（银少）
   3. 数学表示：密度 = k × log（曝光量）
2. **彩色负片**
   1. 除了银像，还有染料像
   2. 蓝→黄、绿→品红、红→青
   3. 还有整体的橙色色罩（color mask）
   4. 色罩是为了改善色彩再现而有意设计的

#### 反转的数学原理

简单的反相（invert）不够，需要正确的色彩校正：

1. **线性反相**
   1. 理想情况：输出 = 1 - 输入
   2. 但实际胶片不是线性的，有对数特性
2. **色罩去除**
   1. 彩色负片有橙色色罩
   2. 需要减去或补偿这个色罩
   3. 可以使用灰场校正，或专门的负片预设
3. **曲线调整**
   1. 胶片有特性曲线（S 曲线）
   2. 需要用曲线工具还原
   3. 暗部和亮部需要不同的调整

### Lightroom 处理流程

#### 基础调整原理

| 滑块               | 作用说明                                                                 |
| ------------------ | ------------------------------------------------------------------------ |
| 曝光（Exposure）   | 整体亮度，线性缩放像素值，类似拍摄曝光但为数字调整                       |
| 对比度（Contrast） | 明暗差异，应用 S 曲线使亮部更亮、暗部更暗                                 |
| 高光（Highlights） | 恢复亮部细节，仅作用于高亮像素（约 80-100%），向左拖动降低高光           |
| 阴影（Shadows）   | 恢复暗部细节，仅作用于暗部像素（约 0-20%），向右拖动提亮                 |
| 白色（Whites）     | 设置最亮点，调整直方图右端点，类似扫描白场                               |
| 黑色（Blacks）     | 设置最暗点，调整直方图左端点，类似扫描黑场                               |

#### 色调曲线原理

曲线是强大的调整工具，理解其原理：

1. **坐标系**：
   1. X 轴：输入亮度（原始值）
   2. Y 轴：输出亮度（调整后的值）
   3. 对角线：线性，无变化
2. **S 曲线**
   1. 暗部向下弯曲：暗部更暗
   2. 亮部向上弯曲：亮部更亮
   3. 结果：增加对比度
   4. 胶片特性就是 S 曲线
3. **反 S 曲线**
   1. 暗部向上弯曲：暗部提亮
   2. 亮部向下弯曲：亮部压暗
   3. 结果：降低对比度，压缩动态范围
4. **单独通道曲线**
   1. 可以分别调整 R、G、B 通道
   2. 用于校正色偏
   3. 红色通道上提：图像偏红
   4. 绿色通道上提：图像偏绿
   5. 蓝色通道上提：图像偏蓝

#### HSL 调整原理

HSL 是调整彩色胶片的核心：

1. **色相（Hue）**
   1. 作用：改变颜色的种类
   2. 原理：在色轮上旋转颜色
   3. 例如：红色可以偏橙或偏紫
2. **饱和度（Saturation）**
   1. 作用：改变颜色的鲜艳程度
   2. 原理：调整颜色与灰色的距离
   3. 饱和度 0：灰度图像
   4. 饱和度过高：颜色不自然，可能溢出
3. **明亮度（Luminance）**
   1. 作用：改变颜色的亮度
   2. 原理：只影响该颜色的亮度，不影响其他颜色
   3. 例如：降低绿色明亮度可以压暗树叶，增加层次

为什么 HSL 对胶片特别重要？

- 不同胶片有不同的色彩响应
- HSL 可以精确调整每种颜色
- 可以模拟或校正特定胶片的色彩特性

### 锐化与降噪原理

#### 锐化原理

锐化的原理是增强边缘对比度：

1. **什么是边缘**
   1. 边缘是亮度急剧变化的地方
   2. 例如：物体轮廓、纹理细节
2. **锐化算法**
   1. USM（Unsharp Mask）是最常用的算法
   2. 步骤：
      - 创建模糊版本（低通滤波）
      - 原图减去模糊版，得到细节（高通滤波）
      - 将细节乘以一个系数（数量）
      - 加回原图
3. **参数解释**
   1. 数量（Amount）：锐化强度，百分比
   2. 半径（Radius）：影响范围，像素数
   3. 细节（Detail）：控制对高频细节的敏感度
   4. 蒙版（Masking）：只锐化边缘，不锐化平坦区域
4. **胶片锐化注意事项**
   1. 胶片本身有颗粒，过度锐化会让颗粒显得不自然
   2. 蒙版功能很重要，只锐化边缘
   3. 建议少量多次，循序渐进

#### 降噪原理

降噪的原理是去除随机噪点，同时保留细节：

1. **噪点类型**
   1. 亮度噪点：颗粒状的灰度噪点
   2. 颜色噪点：彩色的斑点噪点
   3. 胶片数字化后，噪点可能来自扫描或相机高 ISO
2. **降噪算法**
   1. 分析相邻像素的统计特性
   2. 识别随机噪点（与周围差异大但没有规律）
   3. 平滑噪点，但保留边缘（边缘是有规律的变化）
3. **参数解释**
   1. 明亮度（Luminance）：去除亮度噪点的强度
   2. 明亮度细节：保留多少细节
   3. 颜色（Color）：去除颜色噪点的强度
   4. 颜色细节：保留多少颜色细节
4. **胶片降噪注意事项**
   1. 胶片颗粒是特色，不是噪点！
   2. 不要过度降噪，否则会失去胶片感
   3. 可以区分颗粒和噪点：颗粒是均匀的，噪点是随机的

## 暗房放大原理与实践

### 放大的光学原理

暗房放大本质上是一个投影系统：

#### 放大机的光学系统

1. **光源**
   1. 聚光式（Condenser）：使用聚光镜，光线集中
      - 特点：反差高，清晰度高
      - 原理：点光源成像，边缘锐利
   2. 散光式（Diffuser）：使用漫射板，光线柔和
      - 特点：反差低，宽容度好
      - 原理：面光源成像，过渡柔和
   3. 混光式（Condenser-Diffuser）：介于两者之间
2. **底片夹**
   1. 平整地固定底片
   2. 玻璃底片夹可以压平底片，但可能产生牛顿环
   3. 无玻璃底片夹不会产生牛顿环，但需要底片平整
3. **放大镜头**
   1. 将底片的影像投射到相纸上
   2. 焦距选择：通常是底片对角线的 2-3 倍
      - 135 底片（36x24mm）：50mm 镜头
      - 6x6 底片：80-105mm 镜头
   3. 光圈：通常 f/5.6-f/8，最佳光圈
4. **皮腔**
   1. 连接镜头和光源
   2. 可以伸缩，调整放大倍率

#### 放大倍率计算

放大倍率的计算公式：

- 放大倍率 = 影像尺寸 / 底片尺寸
- 或者：放大倍率 = 像距 / 物距
- 物距：底片到镜头的距离
- 像距：镜头到相纸的距离

例如：135 底片（36mm 宽）放大到 8x10 英寸（203mm 宽）

- 放大倍率 = 203 / 36 ≈ 5.6 倍

### 相纸原理

#### 相纸的结构

相纸的结构与胶片类似，但有一些差异：

1. **纸基（或片基）**
   1. 涂塑纸基（RC，Resin Coated）：塑料涂层，防水，干燥快
      - 优点：易处理，适合初学者
      - 缺点：质感不如纸基
   2. 纤维纸基（FB，Fiber Based）：纯纸基，需要更长的水洗和干燥
      - 优点：质感好，收藏级
      - 缺点：处理麻烦
2. **乳剂层**
   1. 黑白相纸：卤化银，通常是氯化银（感光度低）
   2. 彩色相纸：多层乳剂，成色剂
   3. 与胶片相比，晶体更小，颗粒更细
3. **防光晕层**
   1. 吸收散射光，提高清晰度
   2. RC 相纸通常有，FB 相纸可能没有

#### 相纸的反差原理

为什么需要不同反差的相纸？

1. **反差等级**：从 0 到 5，从最软到最硬，从低反差到高反差。
2. **可变反差相纸**
   1. 原理：乳剂层中有两种卤化银，分别感蓝和感绿
   2. 使用不同颜色的滤色片，可以改变有效反差
   3. 黄/品红滤色片：颜色越深，反差越高
   4. 优点：一盒相纸可以获得各种反差，经济灵活
3. **反差选择原理**
   1. 底片反差高 → 选择低反差相纸
   2. 底片反差低 → 选择高反差相纸
   3. 目标：获得合适的最终照片反差

### 放大工艺流程

#### 对焦原理

放大对焦的关键是精确聚焦：

1. **为什么对焦重要**
   1. 放大倍率通常较高，对焦误差会被放大
   2. 轻微的失焦会导致照片不清晰
2. **对焦方法**
   1. 先用白卡纸大致对焦
   2. 使用对焦放大镜（Focus Magnifier）
   3. 放大看颗粒，当颗粒最清晰时，对焦准确
   4. 注意：底片的颗粒是对焦的最佳参考
3. **景深考虑**
   1. 放大镜头的景深很小
   2. 底片不一定完全平整
   3. 最佳对焦在底片中心
   4. 可以稍微收缩光圈（如 f/8），增加景深

#### 试条原理

为什么必须做试条？

1. **曝光预测的困难**
   1. 放大倍率变化
   2. 相纸批次差异
   3. 光源老化
   4. 底片密度差异
   5. 这些因素使得准确预测曝光时间很困难
2. **试条方法**
   1. 取一条相纸，放在相纸位置
   2. 用黑卡纸遮挡，逐段曝光
   3. 例如：第一段 5 秒，第二段 10 秒，第三段 15 秒...
   4. 正常冲洗
   5. 选择最佳的曝光时间
3. **评价试条**
   1. 观察高光：应该有细节，不应该死白
   2. 观察阴影：应该有细节，不应该死黑
   3. 观察整体：反差适中，层次丰富

#### 局部控制原理

遮挡和加光的原理是改变局部曝光：

1. **遮挡（Burn）**
   1. 目的：让某个区域变暗
   2. 方法：用手或工具遮挡该区域，减少曝光
   3. 原理：该区域接收到的光量减少，最终密度降低
   4. 技巧：手要不断移动，避免明显边缘
2. **加光（Dodge）**
   1. 目的：让某个区域变亮
   2. 方法：整体曝光后，单独对该区域额外曝光
   3. 原理：该区域接收到的光量增加，最终密度增加
   4. 技巧：周围也要稍微加光，过渡自然
3. **为什么局部控制有效**
   1. 底片的动态范围可能超过相纸
   2. 局部控制可以压缩或扩展局部动态范围
   3. 类似于数字后期的局部调整，但暗房是光学方式

## 输出与分享

### 文件格式与色彩空间原理

#### 文件格式选择

|      | **TIFF**                 | **JPEG**                             | **PSD**                |
| ---- | ------------------------ | ------------------------------------ | ---------------------- |
| 特点 | 无损压缩，保留所有信息   | 有损压缩，文件小                     | 分层文件，保留编辑历史 |
| 原理 | LZW 压缩，不损失任何数据 | 基于人眼视觉特性，丢弃不太重要的信息 | Photoshop 原生格式     |
| 适用 | 存档、高质量打印         | 网络分享、电子邮件                   | 需要继续编辑的文件     |
| 建议 | 16 位，Adobe RGB         | 质量 80-90，sRGB                     | 仅用于工作文件         |

#### 色彩空间原理



|      | sRGB                   | Adobe RGB          | **ProPhoto RGB**                             |
| ---- | ---------------------- | ------------------ | -------------------------------------------- |
| 特点 | 最小的色彩范围         | 更大的色彩范围     | 最大的色彩范围（需要 16 位支持）             |
| 原理 | 标准红、绿、蓝三原色   | 更饱和的三原色     | 非常饱和的三原色（包括一些人眼看不到的颜色） |
| 适用 | 网络分享、大多数显示器 | 打印输出、专业摄影 | 专业后期、存档                               |
| 优势 | 兼容性最好             | 青色和绿色范围更大 | 后期调整空间最大                             |

### 打印原理

#### 打印技术原理

不同打印技术的原理差异：

1. **喷墨打印**
   1. 原理：微小的墨滴喷射到纸上
   2. 染料墨水：色彩鲜艳，但耐光性差
   3. 颜料墨水：耐光性好，但色彩可能稍差
   4. 艺术微喷（Giclée）：高精度喷墨，专业质量
2. **银盐打印**
   1. 原理：传统暗房，相纸曝光显影
   2. 特点：质感最好，但成本高
   3. 适用：收藏级作品
3. **激光打印**
   1. 原理：静电墨粉
   2. 特点：速度快，但不适合高质量照片

#### 纸张选择原理

1. **表面类型**
   1. 光面（Glossy）：反光，色彩鲜艳，反差高
      - 原理：表面有涂层，墨滴停留在表面
      - 适用：商业、风光
   2. 绒面（Matte）：不反光，质感好
      - 原理：表面粗糙，墨滴渗入
      - 适用：人像、艺术
   3. 绸面（Luster/Satin）：介于两者之间
      - 适用：大多数情况
2. **纸张重量**
   1. 克重（gsm）：每平方米的克数
   2. 越重：质感越好，越不容易卷曲
   3. 建议：170-300gsm 适合大多数照片
3. **无酸纸张**
   1. 原理：与底片保存相同，酸性纸张会导致褪色
   2. 适用：长期保存的作品

掌握本节内容后，你即可根据实际需求选择合适的保存、扫描和后期方案。胶卷后期处理既有技术层面的精确控制，也有创意表达的发挥空间。
