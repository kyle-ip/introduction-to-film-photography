### 过期胶卷使用

#### 胶卷变质

过期变质核心是乳剂化学不稳定：

**潜影衰退**——未曝光卤化银也会自还原形成银簇（灰雾），已形成潜影会氧化消散，感光度下降。

**增感染料分解**——氧化破坏共轭体系、与明胶反应，光谱响应缩小、彩色偏色。

**明胶老化**——水解/氧化导致乳剂脆化、脱落或与卤化银相互作用。

**彩色成色剂变质**——氧化、水解、与明胶交联，色彩平衡破坏（常偏黄、偏品红）。

**影响变质速度的环境因素：**

| 因素 | 加速变质的条件       | 减缓变质的条件 | 原理                         |
| ---- | -------------------- | -------------- | ---------------------------- |
| 温度 | >25°C                | <10°C（冷藏）  | 温度每升高10°C，反应速率×2-4 |
| 湿度 | >60%                 | <40%           | 高湿度加速水解和氧化         |
| 氧气 | 暴露在空气中         | 密封包装       | 氧气参与氧化反应             |
| 射线 | X射线、γ射线、宇宙线 | 铅盒屏蔽       | 高能射线诱发灰雾             |

#### 过期胶卷的特点

感光度下降（十年过期 ISO400 可降至约 100 或更低）、灰雾增加、反差降低、彩色卷色彩偏移（常偏黄/品红，蓝敏层变质最快）、颗粒变粗。上述效果可作特殊风格使用。

#### 感光度下降与曝光补偿

可近似用 ISO_effective ≈ ISO_original × e^(-kt) 估算（k 为衰变常数，t 为过期年数）。下表为经验补偿，品牌/批次/保存差异大，须包围曝光。

| 过期时间 | 黑白胶卷补偿 | 彩色胶卷补偿 | 说明                       |
| -------- | ------------ | ------------ | -------------------------- |
| <1年     | +0-0.5档     | +0-0.5档     | 几乎无影响，正常使用       |
| 1-5年    | +0.5-1档     | +1-1.5档     | 轻微变质，需要补偿         |
| 5-10年   | +1-2档       | +2-3档       | 明显变质，反差降低         |
| 10-20年  | +2-3档       | +3-4档       | 严重变质，灰雾明显         |
| >20年    | +3档以上     | 不推荐使用   | 几乎不可用，或仅作特殊效果 |

**注意**：上表为粗略估计，品牌/批次/保存差异大，须包围曝光。

#### 使用要点

**评估**：看 EXP 日期、包装是否破损受潮、是否曾冷藏。**曝光**：按上表估算补偿，包围曝光（正常/+1/+2）并记录。**显影**：黑白可延长 10-30% 补偿感光下降，勿过度否则灰雾加重；彩色建议送店。**后期**：扫描时调黑/白场减灰雾，彩色需校白平衡。重要拍摄勿用过期卷；先试拍、记数据、接受失败。

### 迫冲和减冲的物理化学原理

迫冲（Push）与减冲（Pull）通过控制显影程度，改变有效感光度、反差与颗粒。下面简述显影动力学与两种技法的原理。

#### 显影动力学简述

显影为多步反应：显影剂扩散至晶体表面→吸附于潜影中心→电子转移（Ag⁺+e⁻→Ag，决速步）→银粒聚集→产物扩散。显影程度用反差系数 γ=ΔD/ΔlogE 表示：正常 γ≈0.7-0.8，迫冲 γ>0.8，减冲 γ<0.7。

#### 迫冲（Push）的原理

用高于标称 ISO 拍摄（曝光不足），延长显影补偿。例：ISO400 按 800 拍（-1 档），高光潜影足、暗部潜影少；正常显影则暗部几乎不显，延长显影可拉高中间调与部分暗部，但暗部密度仍偏低。

**效果原因**：显影速率与潜影中心数量成正比（自催化）。高光显影快、延长后密度大增；暗部显影慢、增加有限→反差提高。延长显影也使银粒生长时间变长、二次成核增多→颗粒变粗；暗部显影剂被高光“竞争”、生长不全→细节损失；未曝光区被显影中心“传染”及自还原→灰雾增加。**实践上**：迫冲即「少曝光、多显影」，用显影时间换有效感光度，代价是反差与颗粒增加、暗部细节减少。

#### 减冲（Pull）的原理

用低于标称 ISO 拍摄（曝光过度），缩短显影。例：ISO400 按 200 拍（+1 档），高光潜影极多；正常显影则高光过曝，缩短显影可限制高光、中间调与暗部仍可达到可用密度。

**效果原因**：高光显影被提前停止、密度受限，暗部仍能完成→反差降低。显影时间短、银粒生长少，且潜影中心多、分布更匀→颗粒更细。减冲过度则高光密度差缩小、细节丢失。**实践上**：减冲即「多曝光、少显影」，适合需要更柔和、细颗粒或高反差场景降反差时使用。

#### 迫冲和减冲的实践

**迫冲（Push）实践：**

| 胶卷  | 原ISO | 迫冲到 | 显影时间增加比例 | 典型显影时间（D-76 20°C）  |
| ----- | ----- | ------ | ---------------- | -------------------------- |
| HP5+  | 400   | 800    | 约30-40%         | 10-11分钟（正常7.5分钟）   |
| HP5+  | 400   | 1600   | 约75-100%        | 14-15分钟                  |
| Tri-X | 400   | 800    | 约30-40%         | 10.5-11分钟（正常7.5分钟） |
| Tri-X | 400   | 1600   | 约75-100%        | 15-16分钟                  |

**举例**：HP5+ 400 按 800 拍（-1 档），D-76 原液正常 7.5 分钟，迫冲 800 需要延长到 10-11 分钟（×1.4）。

**减冲（Pull）实践：**

| 胶卷  | 原ISO | 减冲到 | 显影时间缩短比例 | 典型显影时间（D-76 20°C） |
| ----- | ----- | ------ | ---------------- | ------------------------- |
| HP5+  | 400   | 200    | -20-30%          | 5.5-6分钟（正常7.5分钟）  |
| HP5+  | 400   | 100    | -35-45%          | 4-5分钟                   |
| Tri-X | 400   | 200    | -20-30%          | 5.5-6分钟                 |

**举例**：HP5+ 400 按 200 拍（+1 档），显影时间缩短到 5.5-6 分钟（×0.75）。

**使用场景**：迫冲——弱光需更高快门、要更高反差/颗粒、特殊效果。减冲——要更细颗粒/柔和、高反差场景降反差、误过曝挽救。**注意**：先试冲、记数据；迫冲勿超 2 档、减冲勿超 1 档；适合迫冲的胶卷如 HP5+、Tri-X，T-Max/Acros 等慎用。

### 反转冲洗（黑白正片）

冲洗后直接得到正像，类似彩色反转片。常用工艺：柯达 Technidol、伊尔福 Direct Positive；自配复杂，建议用商品套药。属进阶技术，入门先掌握普通黑白负片。

### 中途曝光（萨巴蒂效应，Solarization）

显影中途做第二次均匀曝光，使影像部分反转并出现轮廓线（麦基线）。过程：一次曝光→部分显影→二次曝光→继续显影→停定水洗。高光区已显影、银多挡光，二次曝光影响小；暗部未显影、卤化银多，二次曝光形成新潜影，再显影后变深→明暗部分反转。边界处因衍射与显影差异形成麦基线。

**关键参数：**

| 参数           | 效果                                        | 推荐值                  |
| -------------- | ------------------------------------------- | ----------------------- |
| 第一次显影时间 | 太短：效果弱；太长：效果过强                | 正常显影时间的30-60%    |
| 第二次曝光量   | 太少：效果弱；太多：过度反转                | 相当于正常拍摄的1-5档   |
| 第二次曝光光源 | 安全灯（红/橙光）：效果温和；白光：效果强烈 | 安全灯，距离1-2米       |
| 第二次曝光时间 | 太短：效果弱；太长：过度反转                | 1-10秒（根据光源强度）  |
| 胶卷类型       | 高反差胶卷：效果明显；低反差胶卷：效果弱    | HP5+、Tri-X等高反差胶卷 |

#### 操作要点

暗房内备显影罐、显影/停显/定影液、安全灯（红/橙光 15-25W）；选高反差胶卷（HP5+、Tri-X），先用废卷练。一次显影至正常时间的 30-60%→倒出显影液（不水洗）、沥干→开安全灯对胶片均匀曝光 1-10 秒（距灯 1-2 米）→关灯、回罐、二次显影，总时间接近或略超正常显影→停定水洗干燥。二次曝光须均匀（可轻移胶片或光源）；总显影勿超正常 120% 以免灰雾过重。

**一次显影时间比例与效果：**

| 第一次显影时间比例 | 效果                         |
| ------------------ | ---------------------------- |
| 20-30%             | 温和，轮廓线细，反转不明显   |
| 30-50%             | 适中，轮廓线明显，部分反转   |
| 50-70%             | 强烈，轮廓线粗，明显反转     |
| >70%               | 过强，可能完全反转、细节丢失 |

| 第二次曝光量 | 效果             |
| ------------ | ---------------- |
| 1档以下      | 弱，轮廓线不明显 |
| 1-3档        | 适中，轮廓线明显 |
| 3-5档        | 强烈，明显反转   |
| >5档         | 过强，可能过曝   |

**技巧**：先试冲、记数据；高反差胶卷与高反差场景效果更好；安全灯比白光易控。**注意**：结果难预测，重要照片勿用；一次显影勿超 60%、二次曝光勿超 5 档；确认安全灯不导致灰雾。

### 直接使用相纸拍摄与反转冲洗

跳过胶片，用**相纸**作为感光材料装入相机或针孔机进行曝光，再经**反转冲洗**得到正像照片。原理：相纸乳剂（卤化银）在相机中曝光形成潜影，经首显形成负像银影，再经漂白、二次曝光（或化学反转）、二次显影将剩余卤化银转为正像，最后定影、水洗得到可观看的正片。**用途**：一次成像风格、大幅单张、针孔/实验摄影。**注意**：相纸感光度很低（约 ISO 1–6），需长时间曝光或强光；画幅受相纸尺寸限制；须在暗房或暗袋中装纸与取纸；反转冲洗步骤与黑白负片正冲类似但针对相纸乳剂，可用专用相纸反转套药或适配的化学反转工艺。属实验性技法，建议先用废相纸试拍与试冲。

### 区域曝光法与显影控制（进阶）

区域曝光法不仅是曝光方法，还可通过控制显影时间来控制照片反差，这是安塞尔·亚当斯的核心技术之一。

#### 曝光与显影的关系

**核心原则**：曝光控制暗部，显影控制高光。

**高反差场景（如正午阳光下的风景）**：问题是光比大，按暗部曝光则高光过曝，按高光曝光则暗部死黑。解决方法是曝光正常曝光加 1 档，使暗部更亮，细节更多；显影缩短显影时间，防止高光过曝。结果是整体反差降低，暗部和高光细节均保留。

**低反差场景（如阴天）**：问题是场景灰蒙蒙，反差小。解决方法是曝光正常曝光减 1 档，使暗部稍暗；显影延长显影时间，提高反差，增加颗粒。结果是照片更有冲击力。

| 场景类型         | 曝光          | 显影            | 效果     |
| ---------------- | ------------- | --------------- | -------- |
| 正常反差         | N（正常曝光） | N（正常显影）   | 正常     |
| 高反差（大光比） | N+1（+1档）   | N-1（缩短显影） | 降低反差 |
| 低反差（小光比） | N-1（-1档）   | N+1（延长显影） | 提高反差 |

#### N-1（减冲，降低反差）

正午阳光下或明暗对比强烈的场景，适合用 N-1。操作要点：曝光比正常多加 1 档（暗部更亮、细节更多），显影时间缩短 20%-30%（高光不过曝、整体反差降低）。例：HP5+，D-76 原液，20°C，正常 7.5 分钟，N-1 约 5.5-6 分钟。

#### N+1（迫冲，提高反差）

阴天、雾天等低反差场景，或需要更强对比、更多颗粒、更戏剧化效果时，可尝试 N+1。操作要点：曝光比正常减 1 档（暗部稍暗），显影时间延长 30%-50%（反差提高、颗粒增加）。例：HP5+，正常 7.5 分钟，N+1 约 10-11 分钟。

#### N（正常）

大多数场景用 N（正常曝光、正常显影）即可。

**实践建议**：每次记录曝光调整与显影时间、结果，逐步建立自己的数据。先从 N（正常）熟悉胶卷，再试 N±1；新技法先用废卷试冲，确认后再用于重要拍摄。

### 倒易律失效详解（Reciprocity Failure）

#### 倒易律的物理基础

曝光量 E = I×t；倒易律指 E 相同时影像密度相同，即光强与时间可互换。成立前提为曝光累积线性：光子作用独立、吸收概率恒定、潜影形成只与总光子数有关。中等曝光时间（约 1/1000 秒～1 秒）内近似成立。

#### 倒易律失效的原理

极短或极长曝光时，胶片反应偏离倒易律，按 E=I×t 计算会曝光不足，即倒易律失效（施瓦茨席尔德效应）。**短曝光（HIRF）**：<1/1000 秒、光强极高，电子-空穴复合多、潜影被氧化，有效感光下降。**长曝光（LIRF）**：>1 秒、光强极低，银原子等不到后续光子即氧化，潜影难稳定。施瓦茨席尔德公式 E=I×t^p（p≈0.7–0.9）描述长曝光：时间延长 10 倍时，光强只需降约 5–8 倍即可维持密度。严重程度与晶体大小、缺陷、化学增感等有关；Acros 100 II 的纳米结构晶体深陷阱多，长曝光表现优异。

| 伊尔福FP4+       | 好       | 好     | +1档     | +1.5-2档 | 通用             |
| 伊尔福HP5+       | 好       | +0.5档 | +1档     | +2档     | 通用             |
| 柯达Tri-X 400    | 好       | +0.5档 | +1档     | +2档     | 类似HP5+         |
| 柯达T-Max        | 好       | 好     | +0.5-1档 | +1.5档   | 现代胶片         |

### 一次成像技术

一次成像：按下快门后相纸吐出，滚轮挤破试剂包，显影/定影液在相纸内流动，数十秒至数分钟内成像；温度影响显影速度（冬慢夏快）。

**化学原理**：相纸内集成感光层（卤化银乳剂）、显影剂、定影剂、染料层与试剂包。过程：曝光形成潜影→挤压破裂试剂包、显影液涂布→显影还原银、成色剂偶合生成染料→染料与二氧化钛迁移至成像层、遮光层渐透明→定影溶解未曝光卤化银。反应速率随温度变化（阿伦尼乌斯）；最佳约 20–25°C，过低偏淡偏冷，过高易色偏/色斑。

#### 相机与相纸规格

**宝丽来**：SX-70（折叠机、SX-70 相纸）、600 系列（常见、600 相纸）、i-Type（新系统、相纸较便宜）；色彩复古。

**富士 Instax**：mini（小画幅）、wide（宽幅）、square（方画幅）；色彩清新、易购，入门推荐。

**撕拉片**：显影后撕开得正片与负片；FP-100C、宝丽来 Type 100 已停产，市面多为过期卷。

**后背改机**：拍立得后背可装于 120 / 大画幅（如哈苏 500、玛米亚 645），结合中画幅画质与即影即有，适合人像试拍。

#### 拍摄技巧

**ISO 与测光**：Instax 标称 ISO 800，宝丽来约 640；机身多为平均测光。高反光/白场景加约 1 档，深色减曝；多数机有±2 档补偿。

Instax 倒易率窄，仅 1/60–1/125 附近接近 ISO 800，偏离需补偿：

- 强光 - EI 1000 - 快门速度 1/125 以上
- 闪光灯 - EI 800 - 快门速度 1/125
- 阳光 - EI 800 - 快门速度 1/60 至 1/125
- 阴天 - EI 640 - 快门速度 1/30 至 1/60
- 室内 - EI 400 - 快门速度 1/8 至 1/30

**光量与闪光**：宽容度约 3–4 档，曝光须尽量准确。闪光有效距离有限（如 Instax mini 约 0.6–2.7 m），过远不足、过近过曝；环境光够可关闪光用自然光。

**距离与构图**：注意最近对焦距离（mini 约 0.6 m、wide 约 0.9 m）；近摄镜可缩短；近距离存在视差，构图略留余量。

**温度**：最佳显影约 20–25°C；冬季可捂暖相纸改善显影与色彩，夏季勿过热（>30°C 易过显、偏深）。

**手动**：中高端机型可调快门/光圈（如 Instax Wide 300、部分宝丽来），用法同传统胶片。

**弱光倒易律**：相纸为银盐材料，快门低于约 1/10 秒时可能出现倒易律失效，建议加 0.5–1 档，1 秒以上可加 1–2 档，具体需试拍确定。

**其他**：显影时勿摇相纸；存放阴凉干燥；未开封可冷藏、用前回温防结露；复杂光线可包围曝光。

## 大画幅摄影基础

本节面向已接触 135/120、想了解大画幅或技术相机核心操作的读者。大画幅通过皮腔与前后组移动实现移轴与倾斜，进而控制透视和景深平面。入门准备中已建议先掌握 135/120 再入大画幅，此处作为进阶扩展。

### 镜头涵盖力（Coverage / Image circle）

移轴或倾斜后，镜头成像圈必须大于底片，否则会出现暗角或切边。大画幅镜头通常标注**成像圈直径**与**涵盖角**；选购时要结合常用画幅与最大移轴/倾斜量，确保有足够余量。涵盖力不足的镜头在极限移轴时画质会下降或边缘发暗。

### 皮腔移轴（平移与倾斜）

#### 平移（Rise / Fall / Shift）

前组或后组相对光轴做**平行移动**（不改变镜头与胶片平面的夹角）。

- **上升/下降（rise / fall）**：在不仰拍的前提下把画面向上/下移，用于建筑时避免楼房向后倾、或把不需要的前景裁掉。
- **左右平移（shift）**：水平方向移轴，用于透视校正或接片。

#### 倾斜与摆动（Tilt / Swing）

前组或后组绕水平轴或垂直轴**转动**，改变镜头平面与胶片平面的夹角。

- **前倾/后倾（tilt）**：绕水平轴转动，改变景深平面的**竖直**方向。
- **左右摆动（swing）**：绕垂直轴转动，改变景深平面的**水平**方向。

tilt 与 swing 用于控制景深平面方向，与沙姆定律配合使用。

#### 前组与后组移轴的区别

- **前组移轴**（镜头板平移/倾斜）：主要改变光路、成像圈覆盖位置与**景深平面方向**，对透视影响较小。常用于控制清晰平面（配合沙姆定律）。
- **后组移轴**（后背平移/倾斜）：改变胶片平面与主体的相对关系，直接影响**透视**与形变。常用于建筑透视校正（如后组 rise/fall 校正垂直汇聚）。

实践中：前组 tilt/swing 多用于景深平面控制；后组 rise/fall/shift 多用于透视校正。不同机型与品牌（林哈夫、骑士、星座等）操作方式略有不同，以说明书为准。

### 皮腔延长与曝光补偿（延长系数）

**原理**：大画幅对焦靠前后组移动拉长或缩短皮腔。**近摄时皮腔拉长、像距大于焦距**。照度与像距的平方成反比，因此同一光圈下到达胶片的光比「无穷远对焦」时更少，必须增加曝光。

**延长系数**：设对焦时的**有效像距**为 d（可从皮腔刻度或实测得到），镜头**焦距**为 f，则：

- **曝光补偿倍数** = (d/f)²
- **补偿档数** = 2 × log₂(d/f)

例如 d/f = 2（像距为焦距 2 倍）时，需增加 2 档曝光。用皮腔上的刻度或尺子量取镜头板到胶片平面的距离作为 d，与镜头焦距 f 代入上式；或查厂商/爱好者整理的「焦距–倍率–补偿表」。若不补偿，近摄会曝光不足。

**测光**：大画幅多用手持测光表（入射或点测）。测光表给出的是无穷远对焦时的曝光；近摄时需在测光结果上按延长系数增加曝光。

### 沙姆定律与霍夫曼定律

#### 沙姆定律（Scheimpflug principle）

当**镜头平面**、**胶片平面**与**被摄体平面**（希望整体清晰的那个平面）三条平面**交于一条直线**时，该被摄体平面上的点会在焦平面上成清晰像，即「清晰平面」与「被摄体平面」重合。

**应用**：在建筑、静物、风光中，被摄体往往是一个**倾斜的平面**（如从下往上看的建筑立面、斜放的产品、斜坡）。若仅收小光圈，景深范围仍大致平行于胶片平面，难以覆盖整块倾斜平面。通过**倾斜镜头板或胶片板**（tilt/swing），使「清晰平面」与倾斜的被摄体平面重合，即可用较小光圈甚至较大光圈使该平面整体清晰，而不必收到极小的 f/22 以下（避免衍射）。

#### 霍夫曼定律（Hinge rule）

霍夫曼定律与沙姆定律配合：当镜头倾斜时，**清晰平面围绕一条通过胶片平面的轴（「铰链轴」）旋转**。这条轴与镜头倾斜轴在胶片平面上的交线有关，可帮助预测清晰平面的走向。实践中：先对焦、再微调 tilt/swing，使毛玻璃上的清晰带与主体平面一致。

#### 倾斜后的景深楔形（Depth of field wedge）

使用 tilt 后，景深不再是前后平行的两个平面，而呈**楔形**：靠近相机的一侧很窄，向远处逐渐变宽。理解这一点有助于判断「倾斜多少合适」以及光圈对可用清晰范围的影响。

### 毛玻璃对焦与片夹装片

**毛玻璃对焦**：大画幅在毛玻璃上直接取景对焦，影像上下左右颠倒。对焦时通常用放大镜（对焦放大镜）检查颗粒清晰度；不同机型取景亮度与对焦便利性不同。

**片夹与装片**：大画幅使用单张页片，需在片夹中装载。在暗袋或暗房中打开片夹、放入页片（药膜面朝正确方向）、插回挡板；拍摄时抽出挡板、曝光、插回挡板、取出片夹。流程与 135/120 装卷不同，需熟悉暗袋操作与片夹结构。

### 移轴镜头（135/120 上的 Tilt-Shift）

佳能 TS-E、尼康 PC-E 等**移轴镜头**把大画幅的移轴、倾斜原理运用到 135/120 上。原理相同（平移校正透视、倾斜控制景深平面），但移轴量与画幅较小；适合建筑、静物、风光中需要透视校正或景深平面控制、又不想上大画幅时使用。

### 小结

大画幅的优势在于：移轴校正透视、用沙姆定律与霍夫曼定律控制景深平面、近摄时注意皮腔延长曝光补偿。选购镜头时需关注涵盖力；使用前组与后组时理解其区别；毛玻璃对焦与片夹装片需单独练习。入门准备中已建议先掌握 135/120，此处作为进阶扩展即可。

## 胶卷摄影作品管理

**工作流**：冲洗 → 晾干 → 无酸袋装袋标注 → 扫描/翻拍 → 命名归档 → 后期（调色、除尘）→ 备份 → 分享/输出。

**命名**：建议 `YYYYMMDD_卷号_序号`（如 `20240214_HP5_001_01.tif`）；文件夹可按年/月或项目分，底片单独存档。

**备份（3-2-1）**：3 份、2 种介质、1 份异地或云端（如本机 + 外接盘 + 云盘）；勿只存一份。

**后期**：扫描后可微调对比度/黑白色场/白平衡等，勿过度；常用 Lightroom（管理+处理）、Photoshop（精细）、Capture One。

**文件夹结构示例**（便于按时间或项目查找）：

- 按年/月：`2024/2024-02/20240214_HP5_001/`，卷内文件 `20240214_HP5_001_01.tif` … `_36.tif`
- 按项目：`项目名/卷号_胶卷类型/`，如 `街头_2024春/HP5_001/`，底片袋与扫描文件同卷编号对应
- 底片实体：单独用无酸袋+盒子按时间或项目存放，袋上标注日期、胶卷、显影信息，与数字文件夹的命名对应，便于日后对照。

## 维护与保养

**相机**：用后擦拭、干燥存放（可放干燥剂），长期不用取出电池。定期检查快门、过片、测光。快门帘勿手摸、勿大力气吹；有灰用气吹轻吹。损坏可找师傅维修，先询价。

**镜头**：气吹除灰（勿过猛）→ 软毛刷轻刷 → 必要时镜头纸/清洁液从中心向外轻擦。勿用衣物或普通纸、勿频繁擦；日常可用 UV 镜。轻微霉斑可请人清洁，严重难除；防霉靠干燥与干燥剂。

**暗房**：显影罐与片芯用后洗净、晾干。药剂按说明保存（棕瓶、满装、避光、标日期）。设备存放于干燥清洁处。
