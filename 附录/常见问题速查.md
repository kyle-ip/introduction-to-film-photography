# 常见问题速查

## 简介

本附录提供「现象 → 可能原因 → 对应章节」的一页式故障速查表，以及拍摄前与相机异常时的检查清单，便于第一次出问题时快速定位。详细原理与操作见主教程及附录相应章节。

## 一、故障速查表

### 拍摄与装卷阶段

| 现象 | 可能原因 | 详见 |
| :--- | :------- | :--- |
| 整卷透明或只拍了几张就没了 | 片头未挂上片轴或齿孔未啮合，胶卷未走动；显影罐漏光 | [拍摄基础](../拍摄基础.md)「装卷与拍摄常见问题」；[黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 底片上有橙色或白色条带、边缘发雾（漏光） | 后盖未关严或密封条老化；装卷/卸卷时非全黑；暗盒接缝进光 | [拍摄基础](../拍摄基础.md)「装卷与拍摄常见问题」 |
| 整卷全黑或部分画面全黑 | 冲洗前胶卷被白光照到；误把定影当显影（先误曝光再冲）；显影罐漏光；拍摄时镜头盖未摘或快门未打开（如电池耗尽） | [拍摄基础](../拍摄基础.md)「装卷与拍摄常见问题」；[黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 齿孔撕裂 | 过片拉力过大或齿孔未对准齿轮 | [拍摄基础](../拍摄基础.md)「装卷与拍摄常见问题」 |
| 折痕、新月形压痕 | 装卷或装轴时胶卷被弯折或指甲压到 | [拍摄基础](../拍摄基础.md)；[黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 黑色树枝状/闪电状纹路 | 静电痕，干燥环境下快速抽拉胶卷 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |

### 冲洗阶段

| 现象 | 可能原因 | 详见 |
| :--- | :------- | :--- |
| 底片一段或整卷全透明 | 未挂上片轴、胶卷未随轴转动；显影罐漏光 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 底片一段或整卷全黑（无影像） | 冲洗前意外曝光（暗袋/显影罐漏光）；误将定影当显影 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 底片密度淡薄（薄） | 显影不足：时间过短、温度过低、药液稀释过度或失效、搅动不足 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 底片密度过浓（厚） | 显影过度：时间过长、温度过高、搅动过剧 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 密度不均匀、条纹或斑块 | 搅动不均匀；药液量不足未淹没胶卷；预湿不充分 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 底片不透明、乳白色（定影不足） | 定影时间太短、定影液稀释错误或失效、温度过低 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 干燥后底片表面水渍 | 最终水洗水质硬；未使用润湿液或润湿液浓度过高 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查 |
| 干燥后底片发黄（随时间加重） | 水洗不彻底，残留定影液（海波）与银反应 | [黑白胶片冲洗配方](黑白胶片冲洗配方.md) 问题排查；主教程「底片保存的化学原理」 |
| 彩色底片色偏（偏黄/偏蓝/偏品等） | 彩显温度偏差（最常见）、彩显液疲劳 | [彩色胶片冲洗配方](彩色胶片冲洗配方.md) 问题排查 |

## 二、拍摄前检查清单

外出拍摄前可按下列项自检，减少「到现场才发现没带/没设好」的情况。

| 检查项 | 说明 |
| :----- | :--- |
| 相机电池 | 若机身或测光需电池，确认有电并备备用电池 |
| 胶卷 | 确认已装入胶卷，型号与 ISO 与拍摄意图一致；备一卷备用卷 |
| 相机 ISO 转盘 | 已按胶卷 ISO 设定（老机多为手动） |
| 测光表 | 若用手持测光表，电池与模式（入射/反射）正确 |
| 记录本与笔 | 便于记张数、光圈、快门、场景，便于冲后复盘 |
| 镜头盖 | 拍摄时记得摘下 |
| 三脚架/快门线 | 若计划长曝光或夜景，提前带上 |

详见主教程 [入门准备](../入门准备.md)「必备附件清单」与「第一卷实操清单」。

## 三、相机异常时检查清单

相机不工作或出片异常时，可按下列顺序排查，避免遗漏常见原因。

| 检查顺序 | 检查项 | 说明 |
| :------- | :----- | :--- |
| 1 | 镜头盖 | 是否未摘下导致「全黑」一张 |
| 2 | 电池 | 电子快门/测光机型无电则快门不动作或测光不工作；换新电池试 |
| 3 | 装卷 | 装卷后回卷旋钮是否随过片转动；未转动则片头可能未挂上，整卷可能透明 |
| 4 | 过片与回卷钮 | 过片扳手是否扳到位；卸卷前是否按下回卷钮再回卷 |
| 5 | 快门速度 | B 门或慢速快门时是否在亮处误触导致过曝 |
| 6 | 测光开关 | 部分机型有测光开关，未开则测光不工作 |
| 7 | 后盖与密封条 | 后盖是否关严、锁扣扣好；密封条是否老化漏光 |
| 8 | 卸卷环境 | 是否在全黑（暗袋/暗房）中卸卷，否则最后几张易漏光 |

若仍无法解决，可查阅相机说明书或送修。更多维护要点见 [进阶内容](../进阶内容.md)「维护与保养」。
