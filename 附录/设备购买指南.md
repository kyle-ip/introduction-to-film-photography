# 设备购买指南

## 一、相机选购决策树

按顺序回答下面问题，根据分支找到对应预算区间的机型，再查文末「按用途与预算机型表」确定具体型号。

### 1. 主要用途

- **日常 / 旅行 / 街拍** → 优先便携、快拍，进入 **2a**
- **人像 / 风光 / 静物** → 优先画质与可控性，进入 **2b**
- **学习摄影基础（快门、光圈、对焦）** → 优先手动、机械，进入 **2c**
- **先试水、随便玩玩** → 直接进入 **2c**，选 135 手动单反即可

### 2. 画幅与形态

**2a 便携 / 快拍**

- 极致便携（能揣兜）→ 旁轴或小型定焦机  
  - 预算 2000 内：奥林巴斯笔式 / 宾得 Auto110 / 胶片傻瓜机  
  - 2000–5000：奥林巴斯 OM-1 / 康泰时 T 系列 / 理光 GR 系列  
  - 5000+：徕卡 M 系列 / 康泰时 G 系列  
- 一般便携（小单反即可）→ 轻便 135 单反  
  - 预算 2000 内：宾得 K1000 / 奥林巴斯 OM-1  
  - 2000–5000：尼康 FM2/FE2 / 佳能 AE-1 Program  
  - 5000+：尼康 F3 / 佳能 F-1  

**2b 画质 / 慢拍**

- 135 即可 → 同「2c」135 推荐，按预算选机身+镜头  
- 要更好画质 → 120  
  - 方形 / 双反：2000 内 海鸥 4A/4B / 雅西卡 Mat-124G；2000–5000 玛米亚 C330/C220；5000+ 禄来 3.5F/2.8F  
  - 645 可换镜头：2000 内 基辅 88 / 富士 GS645；2000–5000 玛米亚 M645 / 勃朗尼卡 ETRS；5000+ 哈苏 500 / 宾得 67  
- 大画幅（4×5 等）→ 星座 Toho、骑士等；需单独学习，新手慎入  

**2c 学习基础 / 试水（135 手动）**

- 纯机械、少依赖电池 → 宾得 K1000 / 奥林巴斯 OM-1 / 尼康 FM（2000 内）；尼康 FM2（2000–5000）；尼康 F3 HP（5000+）  
- 要自动曝光、兼顾手动 → 佳能 AE-1 / AE-1 Program（2000 内）；奥林巴斯 OM-2 / 尼康 FE2（2000–5000）；尼康 F3 / 佳能 A-1（5000+）  

**镜头建议**：入门先配 50mm f/1.8 或 f/2；进阶可加 35mm 或 85mm；120 双反为固定镜头，120 单反多用 80mm 标头。

## 二、选购因素优先级

决策前可先自评优先级，再对照上表：

| 因素     | 说明                     | 自评（勾选或标注）     |
| -------- | ------------------------ | ---------------------- |
| 便携性   | 体积重量、是否常随身     | 便携优先 / 平衡 / 画质优先 |
| 预算     | 机身+镜头可接受总价      | 2000 内 / 2000–5000 / 5000+ |
| 拍摄节奏 | 快拍（街拍 / 日常） vs 慢拍（风光 / 人像） | 快拍 / 平衡 / 慢拍     |
| 对焦     | 手动 vs 自动             | 手动 / 均可 / 自动     |
| 画幅     | 135 / 120 / 大画幅       | 135 / 120 / 大画幅     |
| 快门     | 机械（少用电） vs 电子（更多功能） | 机械 / 无所谓 / 电子   |

**因素速查**：快拍偏便携、旁轴/小单反/傻瓜机；慢拍偏画质与稳定、120/专业 135/大画幅。定焦入门建议 50mm；可换镜头系统扩展性强，固定镜头多更便携。机械快门少依赖电池，电子快门多带自动曝光。

## 三、选购注意事项

1. 购机的前提是对意向机型有充分了解。建议先在网上找一下使用指南，或在 B 站油管看看测评视频。
2. 先买些试机卷以便到手就能测试，要是自己会冲洗就更好，能发现更多问题（比如漏光 / 不合焦之类的，有时只有把照片冲出来才看得出）。
3. 能面交、当面试机最佳。线上购买优先选择描述清晰、同意延长确认时间以试卷的卖家；下单前与卖家确认清楚，收货后拍摄一镜到底的开箱视频（关键细节要拍摄到位），并保留沟通记录。
4. 很多（尤其纯机械结构、无电子元件的）相机只要保养得当都是很皮实耐用的。询问维修情况可了解哪里可能有坑，良心卖家会提醒你注意事项。
5. 老款机型多少都有些毛病，但胶片机的好处是网上资料很多，有时甚至自己动手就能解决（只要不碰冷门机型）。
6. 机身外观成色自行判断即可，一般无严重磕碰、凹陷、变形，饰皮 / 皮腔完整、无大面积开胶或缺失问题都不大。
7. 以下 Checklist 旨在理清有哪些问题。不是说有问题就不能收，结合价格综合考量就行。

### 快门与帘幕

- [ ] 各档快门均可释放（从最快到最慢抽几档试）
- [ ] 不同档位快门声 / 时长有明显区别（如 1/125 与 1/60）
- [ ] B 门：按下打开、松手关闭，无卡死
- [ ] 从镜头座朝里看：帘幕 / 叶片无破洞、撕裂、明显褶皱
- [ ] 快门打开时帘幕 / 叶片动作完整、无半开或卡顿
- [ ] 电子快门机：装电池后各档可释放；机械备用档（若有）在无电时可用

B 站有测试快门速度是否准确的教程，可以自行查阅。

### 测光（有机身测光时）

- [ ] 装规定电池后，测光指针 / LED / 液晶有反应
- [ ] 对准明、暗不同物体时，测光指示随亮度变化
- [ ] 与测光表或测光 App 对比，误差在可接受范围（如 ±1 档内）

### 取景器与对焦屏

- [ ] 取景器内无霉、无脱膜、无大面积划痕
- [ ] 裂像 / 微棱 / 磨砂对焦区域清晰可用
- [ ] 取景亮度正常、无严重发黄发暗
- [ ] 屈光度调节（若有）：可调、无松动或卡死
- [ ] 旁轴：黄斑 / 叠影清晰，无穷远时重合

### 反光镜（单反）

- [ ] 反光镜无破损、无大面积霉与污渍
- [ ] 按下快门后反光镜抬起、回弹到位，无卡顿、异响
- [ ] 反光镜预升（若有）：可正常锁定与释放

### 对焦

- [ ] 无穷远能合焦（取景器 / 黄斑与实拍一致）
- [ ] 最近对焦距离可达到标称值
- [ ] 对焦环从无穷远到最近连续顺畅，无空程、卡点

### 镜头：卡口与安装

- [ ] 卡口无变形、无严重磨损与磕痕
- [ ] 镜头装上后锁紧可靠、无松动、无旷量
- [ ] 光圈联动拨杆 / 触点（若有）：与机身配合正常，收缩光圈正常

### 镜头：光圈与机械

- [ ] 光圈环转动顺畅、档位清晰、无滑档
- [ ] 各档光圈叶片收放正常，无粘连、漏油、不开合
- [ ] 对焦环转动顺畅，阻尼均匀（标尺与无穷远 / 最近对焦一致）
- [ ] 变焦环（若有）：顺滑、无卡顿、无滑移

### 镜头：镜片

- [ ] 前组：无严重划痕、脱膜、霉斑
- [ ] 后组：同上
- [ ] 对光看内部：无大面积灰尘、霉丝、雾状（少量尘可接受；霉斑慎收）
- [ ] 霉斑：白 / 彩色毛状或蛛网状为霉，明显则不建议收
- [ ] 镀膜完整、无大面积崩边、起泡

### 镜头：光轴与成像

- [ ] 镜头筒与卡口无严重磕碰、变形、挤压（光轴歪斜多由撞击导致）
- [ ] 透过镜头看前后镜片：同心圆大致同心，无明显错位、倾斜
- [ ] 若能装机：
  - [ ] 对远处点光源（如路灯）对焦，取景中心与成像位置一致
  - [ ] 实拍试卷看画面四角与中心锐度是否均衡（光轴偏时常见一侧软、或取景与成片偏移）


### 胶片仓、过片与倒片

- [ ] 打开后盖：片仓内干净，无锈蚀、残胶、明显污渍
- [ ] 压片板平整、无毛刺、变形、锈迹
- [ ] 过片扳手 / 旋钮顺畅，无卡顿、打滑、过片不完整
- [ ] 计数器随过片正常递增（或归零正常）
- [ ] 倒片钮 / 杆操作正常，倒片顺畅、片头可完全收回暗盒
- [ ] 后盖密封条（若有）：无老化开裂、脱落
- [ ] 若有试拍卷：片道无划伤片基、无漏光（片边异常亮条等）

### 电池仓与电气

- [ ] 电池仓无漏液、锈蚀；触点完好、接触可靠
- [ ] 规定电池装好后，需用电的功能（测光 / 快门 / 卷片等）正常
- [ ] 自拍（若有）：能完成延时释放
- [ ] 闪光触点 / 热靴（若有）：无锈蚀、变形，接触正常

### 其它与附件

- [ ] 三脚架螺孔：螺纹完好、能拧紧
- [ ] 背带环 / 吊环：无开裂、脱焊
- [ ] 日期后背 / 打印（若有）：显示与打印正常
- [ ] 机身盖 / 镜头盖（若随附）：可正常盖紧、无破裂

### 购买渠道与建议

| 渠道         | 优点               | 缺点                 |
| ------------ | ------------------ | -------------------- |
| 本地相机店   | 可当面按清单检查   | 价格可能偏高         |
| 闲鱼/淘宝    | 选择多、价格灵活   | 需自行鉴别、有风险   |
| eBay         | 机型多、国内少见   | 运费与周期长、售后难 |
| 胶片社群/群组 | 玩家转让、多爱惜   | 货源不稳定           |

**建议**：能当面试机最佳；线上购买优先选支持退换、描述清晰的卖家；下单前与卖家确认「快门、测光、镜头」是否正常，并保留沟通记录。新手第一台可优先考虑保有量大的机型（如佳能 AE-1、宾得 K1000、尼康 FM2、奥林巴斯 OM-1），维修与资料较多。

- 测光表：约 10–15%（若需要）

- 冲洗设备：约 10%
- 胶卷：约 10–15%  

先满足拍摄与冲洗的基本需求，再随使用逐步添置。

## 五、放大机与相纸选购要点（可选）

若打算做暗房放大，可参考下列要点选设备；入门也可先扫底片、再考虑放大。

| 方面 | 说明 |
| :--- | :--- |
| **画幅** | 只放 135 选 135 放大机即可；若放 120 需能换底片夹或选 135/120 两用机 |
| **光源** | 聚光式反差高、锐利；散光式反差柔和、易控；混光式折中。入门散光或混光较易上手 |
| **反差系统** | 可变反差相纸 + 黄/品红滤色片（或专用可变反差滤镜）一盒相纸可调 0–5 号，比固定号数灵活；放大机需能加装滤色片 |
| **放大镜头** | 焦距约等于底片对角线（135 常用 50mm，6×6 常用 80mm）；光圈 f/5.6–f/8 为常用工作光圈 |
| **相纸** | RC 涂塑相纸易处理、水洗快，适合练习；FB 纤维纸基质感好、长期保存更佳，水洗与干燥较费时 |
| **预算与渠道** | 二手放大机与镜头较常见；相纸与药液按主教程「暗房放大」与附录《胶片冲洗工具与操作教程》中的相纸/放大视频链接选型即可 |

## 六、购买渠道简述

- **胶卷**：淘宝/线下器材店；优先近期生产，慎买过期过久的卷。
- **药剂**：淘宝搜「摄影化学药品」或购配好套装；入门建议用成品套装。
- **器材**：见上文「购买渠道与建议」表；重要机身与镜头尽量按「选购注意事项」清单检查。

**重要**：先买一套、用熟再扩展；不必一次买齐所有焦段与附件。
