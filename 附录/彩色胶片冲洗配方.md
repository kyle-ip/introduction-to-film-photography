# 彩色胶片冲洗配方

本附录提供 C-41（彩色负片）与 E-6（彩色反转片）的实操配方、工具清单与问题排查，侧重手冲参考。

彩色冲洗的原理与化学过程见主教程《暗房基础与胶卷冲洗》中「彩色胶片冲洗原理」与「C-41 工艺详解」「E-6 反转片工艺详解」。

## 工具清单

### 核心冲洗设备

与黑白冲洗共用显影罐、片轴、暗袋、引片器、剪刀等，详见附录《黑白胶片冲洗配方》。彩色冲洗对**温度控制**要求更严，设备略有侧重。

| 工具            | 推荐规格 / 备注                                                              | 必要性   |
| :-------------- | :--------------------------------------------------------------------------- | :------- |
| **显影罐**      | 与黑白相同；AP、派森、Jobo 等，135/120 两用罐常用。                          | ⭐⭐⭐ 必备 |
| **显影轴**      | 与罐匹配；C-41/E-6 药液与黑白不同，建议专用轴或冲洗后彻底清洁避免交叉污染。  | ⭐⭐⭐ 必备 |
| **引片器 / 暗袋** | 与黑白相同。                                                                | ⭐⭐⭐ 必备 |
| **精密温度计**  | 量程涵盖 24–40°C，精度 **±0.1°C** 或更好；C-41 彩显需 37.8±0.15°C。          | ⭐⭐⭐ 必备 |
| **量筒 / 量杯** | 按套药说明配备；塑料为宜，避免污染。                                         | ⭐⭐⭐ 必备 |
| **计时器**      | 精度到秒，有响铃；彩色各步时间固定，计时需准确。                             | ⭐⭐⭐ 必备 |

### 温控设备（彩色工艺核心）

C-41 与 E-6 标准温度约 **38°C**，彩显允许偏差仅 ±0.15°C，温控是成败关键。

| 工具              | 推荐规格 / 备注                                                                 | 必要性   |
| :---------------- | :------------------------------------------------------------------------------ | :------- |
| **恒温水浴**      | 可将药液与显影罐维持在 37.8–38°C，精度 ±0.1°C；Jobo、派森等恒温水浴或专业冲机。 | ⭐⭐⭐ 必备 |
| **水浴盆 + 温度计** | 低成本替代：大盆装热水，用温度计调节水温，显影罐浸入保温；夏季可用冰水降温。    | ⭐⭐ 手冲常用 |
| **预热 / 预冷**   | 药液使用前需调至工艺温度；提前 15–30min 将瓶放入水浴。                          | ⭐⭐⭐ 必备 |

夏季室温高时，维持 38°C 比黑白 20°C 更容易；若无法稳定温控，建议送店冲洗。

### 药水与耗材

| 工具           | 推荐规格 / 备注                                                                 | 必要性     |
| :------------- | :------------------------------------------------------------------------------ | :--------- |
| **C-41 套药**  | Tetenal、Fuji Hunt、Kodak Flexicolor、派森、强品等；按说明书配制与保存。        | C-41 必备  |
| **E-6 套药**   | Tetenal、Fuji Hunt、派森等；步骤多、药液种类多，建议送店或使用专业冲机。        | E-6 必备   |
| **稳定液**     | 套药通常包含；提高染料稳定性，不可省略。                                        | ⭐⭐⭐ 必备   |
| **纯净水**     | 配药与最终水洗用蒸馏水或纯净水，避免水垢与杂质影响色彩。                        | ⭐⭐⭐ 必备   |
| **润湿液**     | 柯达 Photo-Flo 或伊尔福 Ilfotol；彩色底片同样需要，防水渍。                     | ⭐⭐⭐ 必备   |
| **药液储存瓶** | 棕色玻璃或 HDPE 塑料瓶，避光、密封、满装减少氧化；彩显液尤其易氧化，开封后尽快使用。 | ⭐⭐ 推荐   |

### 干燥与保存

与黑白相同：胶片夹、无尘干燥空间、无酸底片袋、手套、记号笔等，见《黑白胶片冲洗配方》。彩色底片/正片对高温敏感，干燥时避免暴晒或靠近热源。

## C-41 流程（彩色负片）

### 工艺概要

C-41 是彩色负片标准工艺，绝大多数彩负（柯达、富士等民用卷）通用。彩显温度 37.8±0.15°C 为关键，偏差会导致明显色偏。

### 冲洗配方（标准工艺）

**基础设定**：各步温度 **37.8°C**（漂白、水洗、定影、稳定允许 38±3°C）；具体时间与稀释比以所用套药说明为准。

| 步骤       | 药水           | 温度         | 搅动方式                    | 时间      | 备注                                               |
| :--------- | :------------- | :----------- | :-------------------------- | :-------- | :------------------------------------------------- |
| 1. 预湿    | 清水           | 37.8°C       | 持续搅动                    | 1min（可选） | 使乳剂均匀吸水；部分套药不建议预湿，按说明操作。   |
| 2. 彩显    | 彩显液         | **37.8±0.15°C** | 首 15s 持续，此后每 15s 搅动 5s | **3:15**  | 核心步骤；时间与温度须精确，直接影响密度与色彩。   |
| 3. 漂白    | 漂白液         | 37.8°C       | 每 30s 搅动 5s              | 4:20–6:30 | 将金属银氧化；不同套药时间略有差异。               |
| 4. 水洗    | 流动清水       | 37.8°C       | 持续水流                    | 1:05–3:15 | 去除漂白液残留。                                   |
| 5. 定影    | 定影液         | 37.8°C       | 每 30s 搅动 5s              | 4:20–6:30 | 去除银盐与未曝光卤化银。                           |
| 6. 水洗    | 流动清水       | 37.8°C       | 持续水流                    | 3:15      | 去除定影液。                                       |
| 7. 稳定    | 稳定液         | 24–38°C      | 浸泡，勿剧烈搅动            | 1:05–1:30 | 提高染料稳定性，不可省略。                         |
| 8. 除水渍  | 润湿液         | 室温         | 浸泡 1–2min                 | 1–2min    | 与黑白相同，1+200 比例。                           |
| 9. 干燥    | -              | 室温         | 悬挂晾干                    | -         | 避免高温与直晒。                                   |

### 常用套药

| 品牌           | 说明                                                     |
| :------------- | :------------------------------------------------------- |
| **Tetenal**    | 手冲常用，色彩偏暖；按说明配制，注意彩显液保存期。       |
| **Fuji Hunt**  | 专业级，色彩中性；多供冲印店，小包装较难购得。           |
| **Kodak Flexicolor** | 柯达原厂，需按大包装或分装购买。                 |
| **派森 / 强品** | 国产套药，价格较友好；按说明书稀释与时间操作。           |

搅动方式：倒入彩显液后立即连续翻转 15s，之后每 15s 翻转 5s；漂白、定影可每 30s 搅动一次。具体以套药说明为准。

## E-6 流程（彩色反转片）

### 工艺概要

E-6 是彩色反转片标准工艺，富士 Velvia/Provia、柯达 Ektachrome 等用此工艺。步骤多于 C-41，温控同样严格，自冲难度大，**建议送专业店**或使用 Jobo 等专业冲机。

### 冲洗配方（标准工艺）

| 步骤      | 药水     | 温度         | 搅动方式        | 时间   | 备注                                   |
| :-------- | :------- | :----------- | :-------------- | :----- | :------------------------------------- |
| 1. 首显   | 首显液   | **37.8±0.15°C** | 首 15s 持续，后每 15s 搅动 5s | 6:00   | 形成黑白银负像；时间与温度至关重要。   |
| 2. 水洗   | 流动清水 | 37.8°C       | 持续水流        | 2:00   | 去除首显液，务必彻底。                 |
| 3. 反转   | 反转液   | 37.8°C       | 持续搅动        | 2:00   | 化学反转，使未曝光卤化银可显影。       |
| 4. 彩显   | 彩显液   | **37.8±0.15°C** | 首 15s 持续，后每 15s 搅动 5s | 6:00   | 形成染料正像。                         |
| 5. 调整   | 调整液   | 37.8°C       | 持续搅动        | 2:00   | 为漂白做准备。                         |
| 6. 漂白   | 漂白液   | 37.8°C       | 每 30s 搅动 5s  | 6:00   | 氧化金属银。                           |
| 7. 定影   | 定影液   | 37.8°C       | 每 30s 搅动 5s  | 4:00   | 去除银盐与卤化银。                     |
| 8. 水洗   | 流动清水 | 37.8°C       | 持续水流        | 3:00   | 去除定影液。                           |
| 9. 稳定   | 稳定液   | 37.8°C       | 浸泡            | 1:00   | 提高染料稳定性。                       |
| 10. 干燥  | -        | 室温         | 悬挂晾干        | -      | 避免高温。                             |

### 常用套药

Tetenal、Fuji Hunt、派森等有 E-6 套药；步骤多、药液种类多，自配与保存较复杂。新手建议送店；有黑白与 C-41 经验后再考虑自冲 E-6。

## 问题排查

| 步骤   | 问题现象                     | 可能原因                                                   | 解决方法                                             |
| :----- | :--------------------------- | :--------------------------------------------------------- | :--------------------------------------------------- |
| **彩显** | 底片密度过薄，整体发淡       | 彩显时间不足、温度过低、彩显液失效或稀释错误。             | 检查温度与时间；彩显液开封后尽快使用，勿过度稀释。   |
|        | 底片密度过厚，整体发暗       | 彩显时间过长、温度过高。                                   | 严格按套药时间与温度操作；检查水浴恒温。             |
|        | 整体色偏（偏黄/偏蓝/偏品等） | 彩显温度偏差（最常见）、彩显液疲劳、套药批次差异。         | 彩显温度控制在 37.8±0.15°C；彩显液用后即弃或按说明复用次数。 |
|        | 色彩不均匀、条纹             | 搅动不均匀、药液量不足未完全覆盖胶片、温度波动。           | 规范搅动；保证药液完全浸没胶片；水浴保温。           |
| **漂白** | 底片残留乳白色、不透         | 漂白时间不足、漂白液疲劳。                                 | 延长漂白时间；更换或补充漂白液。                     |
| **定影** | 底片不透明、有云雾           | 定影不足、定影液疲劳。                                     | 延长定影；更换定影液。                               |
| **稳定** | 染料易褪色、保存期短         | 未使用稳定液或时间不足。                                   | 稳定步骤不可省略；按说明时间操作。                   |
| **水洗** | 水渍、斑点                   | 水质过硬、未用润湿液或浓度不当。                           | 最终水洗用蒸馏水/纯净水；润湿液 1+200。              |

更多色彩管理原理见主教程「彩色冲洗的色彩管理」。

| 项目 | 填写 |
| :--- | :--- |

## 工具与流程清单

### 工具准备（C-41）

- [ ] 显影罐、显影轴（建议 C-41 专用或冲洗后彻底清洁）
- [ ] 暗袋、引片器、剪刀
- [ ] 精密温度计（±0.1°C）
- [ ] 恒温水浴或水浴盆 + 温度计
- [ ] 量筒/量杯、计时器
- [ ] C-41 套药（彩显、漂白、定影、稳定）
- [ ] 润湿液、纯净水
- [ ] 药液储存瓶（棕瓶、避光密封）
- [ ] 胶片夹、无尘干燥空间、无酸底片袋、手套、记号笔

### C-41 流程（逐步打勾）

- [ ] **准备**：套药按说明配制；药液预热至 37.8°C；水浴恒温；计时器就位
- [ ] **装卷**：暗袋内装卷，放入显影罐并盖紧
- [ ] **预湿**（若套药建议）：37.8°C 清水 1min，持续搅动，倒出
- [ ] **彩显**：注入彩显液 37.8±0.15°C，3:15，首 15s 持续搅动后每 15s 搅动 5s，到点迅速倒出
- [ ] **漂白**：注入漂白液，按套药时间，每 30s 搅动 5s，倒出
- [ ] **水洗**：流动清水 37.8°C，按套药时间，倒出
- [ ] **定影**：注入定影液，按套药时间，每 30s 搅动 5s，倒出
- [ ] **水洗**：流动清水 37.8°C，3:15，倒出
- [ ] **稳定**：注入稳定液，1:05–1:30，浸泡，倒出
- [ ] **除水渍**：润湿液 1+200 浸泡 1–2min，取出
- [ ] **干燥**：悬挂于无尘处，自然晾干
- [ ] **装袋**：完全干燥后装入无酸底片袋，标注日期、胶卷、工艺

### E-6 流程（简要）

- [ ] 首显 6:00（37.8±0.15°C）→ 水洗 2:00 → 反转 2:00 → 彩显 6:00（37.8±0.15°C）
- [ ] 调整 2:00 → 漂白 6:00 → 定影 4:00 → 水洗 3:00 → 稳定 1:00 → 干燥 → 装袋
