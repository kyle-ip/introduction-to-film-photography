# 胶片摄影术语表

## 基础术语

### 曝光与感光

- **135 胶卷**：35mm 胶卷，最常见，标准 36 张 / 卷（或 24 张）。
- **120 胶卷**：中画幅胶卷，按机型可拍 645/66/67 等不同画幅。
- **ISO**：胶卷感光度，数值越高对光越敏感、颗粒通常越粗。
- **曝光**：光线在胶片上形成潜影的过程；曝光量 = 光强 × 时间。
- **曝光补偿**：在测光基础上人为加减档位，以纠正测光偏差或实现创意效果。
- **包围曝光**：同一场景按正常 / 加一档 / 减一档各拍一张，提高成功率。
- **倒易律**：曝光量相同时（光强×时间相同），理论上密度相同；极短/极长曝光时会失效。
- **倒易律失效**：极短（如 <1/1000s）或极长（如 >1s）曝光时感光效率下降，需增加曝光补偿。
- **DX 码**：胶卷暗盒上的导电条，部分相机可自动识别 ISO 与张数。
- **潜影**：曝光后在卤化银上形成的不可见影像，经显影后变为可见。
- **密度**：底片某处的银或染料多少，决定最终明暗；用 D 表示，特性曲线纵轴。

### 光圈、快门与景深

- **光圈**：镜头内可调节的开孔，控制进光量与景深；f 值越小孔径越大（如 f/2 大于 f/5.6）。
- **快门速度**：快门打开的时间长度，单位秒或分数（如 1/125s）。
- **B 门（Bulb）**：按下期间快门一直打开，松手关闭；长曝光用。
- **T 门（Time）**：按一次打开、再按一次关闭；部分老相机才有。
- **闪光同步速度**：能配合闪光灯使用的最高快门（如 1/125s），更高速度会局部被遮挡。
- **闪光指数（GN）**：GN = 光圈 × 距离（某 ISO 下，常为 ISO 100）；手动闪光时 光圈 = GN ÷ 距离。胶卷 ISO 不同时有效 GN ≈ GN_100 × √(ISO/100)。详见主教程「拍摄基础」闪光灯基础原理。
- **景深**：画面中看起来清晰的纵深范围；光圈越小、焦距越短、距离越远，景深越大。
- **景深预览**：收缩光圈到拍摄档位预览景深，部分单反有此按钮。
- **超焦距**：对焦到无穷远时，景深近界到无穷远都清晰，该近界距离即超焦距；若改为对焦到超焦距，则从超焦距的一半到无穷远都清晰，可获得该光圈下最大景深。详见主教程「拍摄基础」中光圈与景深、估焦小节。
- **估焦（Scale focusing / Zone focusing）**：不通过取景器对焦，而是根据距离刻度或景深表预设对焦距离，使一定范围落在景深内，适合街头快拍、无测距相机或盲拍。常与景深表、超焦距配合使用。详见主教程「拍摄基础」中手动对焦技巧。

### 测光与曝光方法

- **测光表**：测量光线强度的仪器，可机内或手持。
- **18% 灰**：反射式测光的基准，相当于中性灰。
- **阳光十六法则（Sunny 16）**：无测光时的估曝口诀——晴天正午顺光下光圈 f/16、快门 1/ISO（如 ISO 100 用 1/125s）；阴天约 f/8、室内窗边约 f/4 等可类推。详见主教程「拍摄基础」中测光与曝光。
- **区域曝光法**：安塞尔·亚当斯提出的曝光与显影控制方法，将场景亮度分成若干区（如 N、N+1、N-1），用曝光定暗部、用显影控高光。
- **点测光**：只测画面中很小区域；中央重点/平均测光则考虑更大范围。

### 画质与影调

- **颗粒**：底片上的银粒或染料团，高 ISO、过度显影会变粗。
- **反差**：明暗差异程度；高反差黑白分明，低反差偏灰。
- **宽容度**：胶片能同时记录的最亮与最暗的范围，超出则死白或死黑。
- **特性曲线**：密度 D 随曝光量对数 log E 变化的曲线，描述胶片的感光与反差特性。

## 画幅与器材

- **135**：35mm 全画幅，约 24×36mm。
- **半格**：135 胶卷拍 18×24mm，一张胶卷拍两张，张数翻倍。
- **120 画幅**：645（约 56×42mm）、6×6、6×7 等，画幅越大画质通常越好、机身越重。
- **大画幅**：4×5、8×10 英寸等，单张片夹、毛玻璃对焦、移轴与皮腔控制。详见主教程「进阶内容」大画幅摄影基础。
- **毛玻璃**：大画幅/双反等机型的对焦屏，成像上下左右颠倒。
- **移轴**：大画幅或移轴镜头可平移（rise/fall、shift）与倾斜/摆动（tilt、swing），用于透视校正与控制景深平面方向；详见主教程「进阶内容」大画幅摄影基础。
- **镜头涵盖力**：镜头成像圈直径与涵盖角；移轴或倾斜后需成像圈大于底片，否则暗角或切边。
- **皮腔延长系数**：大画幅近摄时皮腔拉长、像距增大，照度与像距平方成反比，需增加曝光；补偿倍数 = (像距/焦距)²，或补偿档数 = 2×log₂(像距/焦距)。
- **沙姆定律（Scheimpflug）**：当镜头平面、胶片平面与被摄体平面三面交于一直线时，该被摄体平面整体清晰；通过倾斜镜头或后背可使「清晰平面」与倾斜的被摄体平面重合，用于建筑、静物等。
- **霍夫曼定律（Hinge rule）**：镜头倾斜时，清晰平面围绕一条通过胶片平面的「铰链轴」旋转；与沙姆定律配合，可预测清晰平面走向。
- **卡口**：机身与镜头的接口规格，如 Nikon F、Canon FD、M42 等，不同卡口一般不通用。
- **标头**：焦距约等于画幅对角线，135 约 50mm，120 约 80mm；视角接近人眼。
- **广角 / 中焦 / 长焦**：焦距短于标头为广角，长于标头为长焦，介于其间可称中焦。
- **取景器**：目镜中看到的画面；旁轴与单反结构不同，单反所见即所得。
- **视差**：旁轴取景与镜头视轴不重合，近距离时取景与实际画面有偏差。
- **黄斑对焦**：旁轴相机中重叠像对焦，两像重合即合焦。
- **裂像 / 微棱**：单反对焦屏上的辅助对焦方式，裂像线对齐或微棱区不闪烁即合焦。

## 暗房术语

### 冲洗流程

- **显影**：用显影液将潜影还原为可见金属银（黑白）或同时生成染料（彩色）。
- **停显**：酸性液中和显影液，迅速停止显影。
- **定影**：溶解未曝光的卤化银，使影像不再感光、可见光。
- **水洗**：洗掉残留药剂，防止日后变色或损坏。
- **预湿**：显影前用清水短暂浸泡，使乳剂均匀吸水，多用于彩色或部分黑白流程。
- **漂白**：彩色工艺中去除金属银、只保留染料影像的步骤。
- **稳定**：彩色工艺最后一步，保护染料、延长保存。

### 药剂与配方

- **显影液**：含还原剂等，将潜影变为可见影像；常用 D-76、ID-11、HC-110 等。
- **D-76 / ID-11**：经典黑白显影液，成分为对苯二酚+米吐尔等，通用性好。
- **F-5 / 定影液**：硫代硫酸盐等，溶解未感光卤化银。
- **停显液**：稀醋酸等，中和显影液。
- **坚膜**：某些定影或单独步骤，加强乳剂机械强度，高温冲洗时常用。
- **C-41**：彩色负片标准工艺，需严格温控（如 38°C）。
- **E-6**：彩色反转片标准工艺，步骤多于 C-41。

### 设备与操作

- **显影罐**：遮光容器，内装片芯，用于装卷与注入药液。
- **片芯**：显影罐内卷胶卷的轴，有不锈钢与塑料等。
- **装卷**：在暗处或暗袋中将胶卷卷到片芯上再放入显影罐。
- **安全灯**：暗房用灯，波长对所用胶片/相纸不敏感（如暗房用红/橙灯；全黑冲洗则不用灯）。
- **灰雾**：未曝光区域不应有的密度，由漏光、过期、显影过度等引起。

### 特殊技法

- **迫冲（Push）**：按高于标称 ISO 拍摄（曝光不足），延长显影时间补偿；反差与颗粒增加。
- **减冲（Pull）**：按低于标称 ISO 拍摄（曝光过度），缩短显影；反差与颗粒降低。
- **N+1 / N-1**：区域曝光法中，N+1 为延长显影提高反差，N-1 为缩短显影降低反差。
- **Gamma（γ）**：特性曲线直线段斜率，表示反差；负片常用 γ≈0.7–0.8。
- **反转冲洗**：黑白或彩色工艺，直接得到正像（幻灯片效果）。
- **中途曝光 / 萨巴蒂效应**：显影中途进行二次均匀曝光，造成部分反转与轮廓线（麦基线）。
- **麦基线（Mackie Line）**：萨巴蒂效应在明暗交界处形成的轮廓线。

## 后期术语

### 数字化

- **扫描**：用扫描仪将底片转为数字文件。
- **透扫**：透射扫描，光线穿过底片由传感器接收；平板扫描仪需透扫适配器。
- **翻拍**：用数码相机对底片或照片拍摄，替代扫描。
- **底扫**：专用于底片的扫描仪，通常分辨率与动态范围优于普通平板透扫。
- **平板扫描**：平板扫描仪加透扫灯/片夹扫描底片。
- **分辨率 / dpi**：每英寸点数，表示扫描精度；常用 2400–4800 dpi 扫 135。
- **ICE（自动除尘）**：部分扫描仪用红外通道识别灰尘并软件去除，仅适用于染料型底片，银盐底片慎用或禁用。

### 文件与调色

- **TIFF**：无损位图格式，适合存档与后期。
- **RAW**：相机/扫描仪的原始数据格式，保留更多信息供后期处理。
- **负片反转**：将负像转为正像（明暗、色彩反转），扫描或后期中完成。
- **色阶**：调整黑场、白场与中间调，控制明暗分布。
- **直方图**：像素亮度分布图，用于判断曝光与调色。
- **白平衡**：校正色温，使白色不偏色；彩色底片扫描后常需调整。
- **色偏**：整体偏向某一颜色，需在后期校正。
- **锐化**：增强边缘清晰度，扫描后常轻微使用。
- **降噪**：减少颗粒/噪点，可能损失细节。
