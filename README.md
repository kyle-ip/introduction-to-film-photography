# 从零开始的胶片摄影 🎞️

一份面向初学者的胶片摄影教程，从器材选择、拍摄、冲洗到后期，覆盖完整流程。注重原理与实操结合，便于建立可复盘的拍摄与冲洗习惯。

胶片摄影是需要耐心和实践的过程。不要害怕犯错，每一次失误都是学习的机会。享受从拍摄到看到最终成品的完整过程，正是胶片摄影的魅力所在。

## 如何使用本教程

建议按顺序阅读：**入门准备 → 拍摄基础 → 暗房基础与胶卷冲洗 → 胶卷后期处理**，再按需翻阅**进阶内容**与**附录**。附录提供设备选购、冲洗配方、术语表、安全须知及扩展资源，可随时查阅。
